# FunInvestAgent

재테크 의사결정을 위한 파트너형 에이전트 엔지니어링 프로젝트입니다. 목표는 단순 종목 추천이 아니라 사용자의 재무 상황, 목표, 위험 감내도, 시장 데이터, 법률 및 규제 리스크를 함께 검토하여 실행 가능한 전략 메모와 보고서를 만드는 것입니다.

## 구조

```text
FunInvestAgent/
  funmemory/   규칙, 지식 베이스, 법률/출처 레지스트리, 도메인 메모리, 데이터 스키마
  funagent/    에이전트 역할, 워크플로우, 스킬, 판단 규칙, 검증 도구
  funoutput/   세션 로그, 리포트, 전략 메모, 법률 리스크 메모, 산출물 템플릿
```

## 핵심 설계

1. 사용자 상황을 먼저 수집한다.
2. 자산군별 시장 데이터를 공식 또는 신뢰 가능한 출처로 확인한다.
3. 수익 기대보다 손실 가능성, 유동성, 세금, 법률 리스크를 먼저 점검한다.
4. 포트폴리오 전략은 단일 정답이 아니라 보수, 기준, 공격 시나리오로 나눈다.
5. 모든 산출물은 출처, 가정, 최신성, 검증 체크리스트를 포함한다.

## 주식 전략 오케스트레이터

주식 전략은 `funagent/skills/stocks/stock_orchestrator.md`에서 조립합니다. 단타와 가치투자를 분리해 현재 행동, 자금 배분, 손절/논리 훼손 조건, 성과 검토 일정을 함께 설계합니다.

모든 주식 전략 리포트는 Verify 평점 계약을 포함합니다. 입력 적합성, 데이터 신뢰도, 분석 논리, 전략 완성도, 리스크 통제, 실행 가능성, 성과 검토성, 표현 안전성을 반복 평가하고 기준 미달 시 수정 또는 보류합니다.

## 빠른 검증

```powershell
python funagent/validators/validate_output.py --root .
```

이 검증기는 필수 폴더, 핵심 템플릿, 출처 최신성 필드, 보고서 필수 섹션 존재 여부를 점검합니다.

## 책임 범위

이 프로젝트는 투자 판단을 자동 위임하기 위한 시스템이 아닙니다. 개인별 투자, 세무, 법률 판단은 관련 자격을 가진 전문가 검토가 필요할 수 있습니다.
