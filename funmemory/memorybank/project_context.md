# 프로젝트 컨텍스트

## 제품 비전

FunInvestAgent는 재테크를 게임처럼 가볍게 소비하게 만드는 서비스가 아니라, 반복 사용을 통해 사용자의 판단력이 좋아지는 고신뢰 재무 파트너다. 재미 요소는 수익률 과장이나 투기 유도 대신 목표 달성률, 리스크 감소, 현금흐름 개선, 학습 진척도에 둔다.

## 주요 사용자

- 재테크를 시작했지만 어디서부터 관리해야 할지 모르는 개인.
- 이미 주식, 부동산, 예금, 연금 등이 흩어져 있어 통합 점검이 필요한 사용자.
- 특정 투자 기회가 왔을 때 법률, 세금, 리스크 체크리스트가 필요한 사용자.
- 시장 상황을 주기적으로 정리해 의사결정 루틴을 만들고 싶은 사용자.

## 성공 기준

- 사용자가 자신의 투자 가능 자금과 위험 한도를 더 명확히 안다.
- 보고서가 특정 상품 홍보가 아니라 의사결정 근거와 보류 조건을 제공한다.
- 법률, 세금, 규제 리스크를 초기에 드러낸다.
- 매주 또는 매월 반복 사용 가능한 워크플로우가 있다.

## 차별화 아이디어

- 투자 성과뿐 아니라 `위험 회피로 지킨 금액`, `고금리 부채 절감 효과`, `리밸런싱으로 낮춘 변동성`을 보상 피드백으로 보여준다.
- 포트폴리오를 RPG 파티처럼 역할화한다. 예: 방어형 현금, 안정형 채권, 성장형 주식, 인플레이션 방패형 현물, 장기 거점형 부동산.
- 사용자의 행동을 퀘스트화한다. 예: 비상금 6개월 채우기, 대출 금리 재협상 확인, 투자설명서 위험 문구 3개 찾기.
