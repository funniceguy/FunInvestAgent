# funmemory

`funmemory`는 FunInvestAgent의 장기 기억 계층입니다. 에이전트가 반복적으로 참고해야 하는 운영 규칙, 시장 도메인 지식, 법률 및 규제 출처, 데이터 스키마, 프로젝트 의사결정 로그를 관리합니다.

## 하위 구조

```text
funmemory/
  domains/       주식, 부동산, 현물, 채권 등 자산군별 분석 프레임
  memorybank/    프로젝트 맥락과 의사결정 이력
  rules/         에이전트 원칙, 법률/컴플라이언스 가드레일
  schemas/       사용자 프로필, 포트폴리오, 시장 관찰, 리포트 스키마
  sources/       공식 출처 레지스트리와 갱신 프로토콜
```

## 업데이트 원칙

- 시장 가격, 금리, 세율, 법령, 공시, 규제 변경 가능성이 있는 항목은 보고서 생성 당일 재확인한다.
- 출처는 공식 기관, 거래소, 공시 시스템, 원문 법령, 1차 데이터 제공자를 우선한다.
- 분석용 메모와 법률적 결론을 분리한다. 법률 메모는 이슈 식별과 확인 질문까지가 기본 범위다.
- 주식 전략은 `domains/stocks.md`와 `domains/stock_strategy_playbook.md`를 함께 사용해 단타와 가치투자 판단을 분리한다.
- 주식 심리전은 `domains/stock_investor_psychology_playbook.md`를 사용해 정보, 기대치, 포지션, 가격 반응, 수급 지속성을 해석한다.
