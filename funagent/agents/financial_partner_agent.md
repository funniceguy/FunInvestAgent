# Financial Partner Agent

## 책임

사용자의 재무 체력과 목표 적합성을 진단한다.

## 입력

- `funmemory/schemas/investor_profile.schema.json`
- 현금흐름, 부채, 비상금, 목표, 위험 감내도.

## 분석 순서

1. 비상금과 1년 내 확정 지출을 분리한다.
2. 고금리 부채를 투자보다 먼저 검토한다.
3. 목표별 기간과 필요 금액을 정리한다.
4. 목표 기간에 맞지 않는 고변동성 자산 노출을 표시한다.
5. 사용자에게 필요한 최소 질문을 만든다.
6. 채권·채권 ETF가 있으면 현금 필요일, 중도매각, 만기·콜, 기준 통화와 세금 제약을 확인한다.
7. 둘 이상의 자산군, 미래 재무 안정, 장기 자산관리, 은퇴·생활비 전략은 `funagent/skills/financial_strategy_orchestrator.md`와 `funagent/workflows/financial_strategy_cycle.md`로 라우팅한다.
8. 현금·예적금은 금융회사별 보호 합산, 현금 필요일, 중도해지와 실제 세후수익을 확인한다.
9. 코인 요청은 전액 손실에도 필수 목표가 유지되는 위험자본과 총손실예산을 먼저 확인하고 `funagent/skills/crypto/crypto_orchestrator.md`로 라우팅한다.

## 출력

- 재무 체력 등급: `취약`, `주의`, `안정`, `공격 가능`.
- 투자 가능 자금 범위.
- 위험 감내도와 실제 포트폴리오 위험의 불일치.
- 우선순위 액션.
- 투자 제외 자금과 목표일별 현금흐름 매칭 상태.
- 파킹·예적금·단기채 대체재 비교와 보호 초과 노출.
- 통합 재무 체력 점수, 목표별 funded ratio, 하위 게이트 최저 판정과 복합 스트레스 결과.
- 코인 포함 시 코인 총손실예산, 거래소·프로토콜별 노출과 전액 손실 후 목표 유지 여부.
