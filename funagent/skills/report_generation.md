# 스킬: 리포트 생성

## 목적

분석 결과를 사용자가 바로 읽고 실행 여부를 판단할 수 있는 보고서로 만든다.

## 절차

1. 보고서 유형을 선택한다.
2. `funoutput/templates`의 템플릿을 복사해 사용한다.
3. 입력 데이터와 가정을 먼저 기록한다.
4. 시장 관찰과 전략을 연결한다.
5. 리스크와 법률/세금 확인 사항을 분리한다.
6. 검증 체크리스트를 완료한다.

## 산출물

- 세션 요약.
- 주간 시장 보고서.
- 포트폴리오 리뷰.
- 전략 메모.
- 법률 리스크 메모.
- 주식 전략 리포트.
- 주식 전략 성과 검토.
- 채권 전략 보고서.
- 주식·채권 통합 장기 포트폴리오 보고서.
- 현금성·예적금·채권 대체재 비교 보고서.
- 통합 재무 전략 보고서.
- 코인 전략 보고서.
