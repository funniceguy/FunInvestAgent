# 스킬: 포트폴리오 진단

## 목적

사용자의 현재 자산, 부채, 현금흐름을 입력받아 재무 체력과 포트폴리오 리스크를 진단한다.

## 입력

- 투자자 프로필.
- 자산 목록.
- 부채 목록.
- 월 현금흐름.
- 목표와 기간.

## 절차

1. 자산과 부채를 표준 스키마로 정리한다.
2. 유동성 버킷을 분류한다.
3. 자산군 비중을 계산한다.
4. 집중 리스크와 부채 리스크를 표시한다.
5. 목표 기간과 자산 변동성을 비교한다.
6. 우선순위 액션을 만든다.

## 산출물

- 재무 체력 등급.
- 자산군 비중.
- 리스크 플래그.
- 리밸런싱 후보.
