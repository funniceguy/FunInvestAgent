# 스킬: 목표 기반 재무 버킷 설계

## 목적

하나의 수익률 목표를 목표별 날짜·현금흐름·손실 허용 범위로 분해하고 적합한 자산 슬리브를 연결한다.

## 버킷

- 생존: 비상금, 보험 공백, 필수 운영자금.
- 확정지출: 3년 이내 주거·교육·세금·대형 지출.
- 중기목표: 3~7년 목표. 손실 후 회복기간이 목표일을 침해하지 않아야 함.
- 장기성장: 7년 이상 적립·복리·은퇴 성장.
- 인출·생활비: 은퇴·정기 현금흐름과 인출 순서 위험.
- 전술기회: 전체 손실예산 내 작은 비중, 코어 목표와 분리.

## 설계 규칙

- 각 목표에 현재 금액, 미래 필요금액, 월 납입·인출, 물가, 세후 필요수익률, 부족액을 기록한다.
- 필요수익률이 현실적 자산 범위를 초과하면 위험을 높여 맞추지 않고 날짜·납입·목표액·인출을 재설계한다.
- 생존·확정지출은 `cash_deposit_orchestrator`의 보호·유동성 판정을 우선한다.
- 중기·장기는 주식·채권·현금성 하위 게이트를 통과한 자산만 사용한다.
- 목표일이 가까워지면 예정된 디리스킹 경로를 사용하고 단기 시장 전망으로 자산배분을 급변시키지 않는다.
- 슬리브 간 동일 종목·지수·발행자·통화·금리·신용 노출을 합산한다.

## 출력

- 목표별 funded ratio, 필요 세후 실질수익률과 현실성 판정.
- 자산 역할·목표 비중 범위·허용 밴드·디리스킹 경로.
- 자동 납입·분배 재투자·인출 순서와 부족액 조정 순서.
- 목표 훼손·중단·재설계 조건.
