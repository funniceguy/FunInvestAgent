# 스킬: 코인 입력·자본 생존 게이트

## 목적

코인 투자 전에 생활과 목표를 훼손하지 않는 위험자본, 총손실예산, 강제매도 가능성을 확정한다.

## 필수 입력

- 비상자금, 3년 내 확정지출, 고비용 부채, 월 잉여현금흐름.
- 투자기간, 목표, 기준통화, 세금 관할, 최대 감내·감당 손실.
- 전체 금융자산, 현재 코인 비중, 거래소·지갑·프로토콜별 노출.
- 사용 가능한 거래시간, 수면시간, 보안 숙련도, 키 복구 계획.

## 계산

- `crypto_loss_budget = min(사용자 허용손실, 목표를 훼손하지 않는 위험자본)`.
- 거래 크기는 `거래 손실예산 / 스트레스 손실률` 이하로 제한한다.
- 스트레스 손실률은 현물 급락, 슬리피지, 거래소 장애, 디페그, 스마트컨트랙트 손실 중 적용 가능한 최악의 비보장 손실을 사용한다.
- 낙폭 `d`의 회복 필요수익률은 `d / (1-d)`로 표시한다.

## 판정

- `clear`: 투자 제외 자금과 총손실예산이 확인되고 전액 손실에도 필수 목표가 유지됨.
- `conditional`: 금액 일부가 unknown이거나 보관·운영 역량 보완이 필요함. 관찰·교육·소액 테스트만 허용.
- `watch_only`: 비상금·부채·확정지출 또는 손실예산 미확인. 신규 주문 금지.
- `blocked`: 생활자금, 차입, 손실복구, 허용손실 초과 또는 강제매도 위험.

## 중대 차단

- 대출·카드론·미수·마진으로 코인 투자.
- 전액 손실 시 비상금, 주거, 의료, 세금, 부채상환 목표 훼손.
- 총 코인 노출과 거래소·프로토콜별 노출을 모른 채 추가 매수.
- 최대 손실액과 수면 중 대응 규칙 없이 단기·파생 전략 실행.
