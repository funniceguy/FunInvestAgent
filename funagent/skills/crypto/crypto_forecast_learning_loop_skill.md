# 스킬: 코인 예측·전략 학습 루프

## 목적

24시간 시장의 방향 예측, 주문, 보관, 수익 전략을 분리 채점하고 반복 실패를 다음 전략의 차단 조건으로 강화한다.

## 예측 계약

- 모든 예측은 `prediction_id`, 원본 작성시각, 대상·거래소·통화, 기간, 상승/횡보/하락 확률 합계 100%, 예상 범위, 검증 시각, 성공·실패 기준을 동결한다.
- 가격 방향, 실행 손익, 손실예산 준수, 출금·보관 안정, 벤치마크 대비를 별도로 채점한다.
- 검증 시각이 지나 결과가 없으면 `matured_unscored`, ID가 없으면 `legacy_unscorable`로 남긴다.

## 복기

1. `funoutput/reports`, `funoutput/reviews`, 코인 예측 원장을 전수 확인한다.
2. 시장 국면, 자산군, 보유기간, 현물/파생/수익 전략으로 표본을 분리한다.
3. 성공, 실패, 미채점, 실행 안 됨, 규칙 위반을 구분한다.
4. 데이터 지연, 거래소 편향, 온체인 라벨, 포지션 혼잡, 잠금해제, 보관·출금, 심리·수면 실패를 원인별 기록한다.
5. 같은 실패가 2회면 관련 최소 점수를 3점, 3회면 5점 높이거나 중대 차단으로 승격한다.

## 통계 계약

- 방향 확률은 Brier score와 보정 구간으로 평가한다.
- 성숙·채점 표본 20개 미만은 적중률 개선이나 규칙 완화를 주장하지 않는다.
- 절차 점수와 실제 수익률을 합치지 않는다. 수수료·세금·가스·펀딩·슬리피지 후 성과를 사용한다.
