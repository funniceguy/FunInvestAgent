# 스킬: 채권 공식 정보 검색

## 목적

금리·곡선·인플레이션·신용·상품 구조·거래 데이터를 최신 공식 또는 1차 출처에서 수집하고 출처별 역할과 한계를 기록한다.

## 검색 순서

1. 중앙은행 정책결정, 회의 일정, 전망과 경제통계.
2. 재무당국·국채당국의 발행계획, 입찰결과, 명목·실질 수익률곡선.
3. 거래소·자율규제기관의 체결가격, 거래량, 수익률과 유동성 자료.
4. 발행사 공시, 투자설명서, 신탁계약·사채관리계약, 신용평가 원문.
5. ETF·펀드의 최신 투자설명서, 월간 운용보고서, 보유내역, 듀레이션, 비용, 분배 공시.
6. 세금·규제는 관할 세무·감독기관 최신 원문. 확정 해석이 필요하면 전문가 확인 대상으로 분리한다.

## 필수 출처군

- 한국: 한국은행 ECOS·통화정책 자료, 국고채 통합정보, KRX 채권 데이터, 금융감독원 DART, 금융투자협회 채권정보.
- 미국: Federal Reserve, U.S. Treasury·TreasuryDirect, SEC EDGAR, FINRA Fixed Income·TRACE.
- 글로벌: 해당 중앙은행·재무당국·거래소·발행사 원문. 보조 데이터는 공식 자료와 교차검증한다.

## 최신성 계약

- 가격·수익률·환율·스프레드: 전략 기준 거래일, 지연 여부와 조회 시각 포함.
- 정책·입찰·이벤트: 발표일, 효력일, 다음 일정 포함.
- 재무·등급·펀드 보유내역: 문서 기준일과 이후 중요 공시 확인.
- 현재 자료를 얻지 못하면 숫자를 추정하지 않고 `unknown`으로 남기며 실행 판정을 낮춘다.

## 출력

- 검색 커버리지와 누락 목록.
- `source_id`, URL, 문서명, 발행일, `data_as_of`, `checked_at`, 1차/보조 구분.
- 충돌 자료와 채택 이유.
- 다음 갱신 시각과 유효기간.
