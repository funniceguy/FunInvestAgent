# 스킬: 채권 금리 국면·수익률곡선 분석

## 목적

정책금리, 명목·실질 수익률곡선, 물가, 성장, 수급과 신용 스프레드를 시나리오로 연결해 듀레이션·신용 노출의 방향과 손익 범위를 만든다.

## 분석 축

- 정책: 현재 금리, 중앙은행 반응함수, 다음 회의와 시장 기대의 차이.
- 곡선: 단기·중기·장기 금리, 기울기와 곡률, 최근 변화. 단일 스프레드로 전체 곡선을 대표하지 않는다.
- 실질·물가: 실질금리, 기대인플레이션, 실제 물가와 에너지·임금 위험.
- 신용: 등급별 OAS·부도·강등·발행 수급. 국채 금리와 신용 스프레드 변화를 분리한다.
- 수급: 국채 발행·입찰, 중앙은행 보유 변화, 펀드 자금, 헤지 비용과 환율.

## 필수 시나리오

- 평행 +100bp / -100bp.
- 단기 중심 상승·하락, 장기 중심 상승·하락, 스티프닝·플래트닝.
- 국채금리 불변 + 신용 스프레드 확대.
- 주식 하락 + 금리 상승 + 환율 불리의 복합 충격.

## 규칙

- 전망은 기본·강세·약세 확률과 트리거, 무효화 조건으로 표현한다.
- 선도금리나 선물가격은 시장 기대이며 확정 예측이 아니다.
- 장기금리를 정책금리만으로 설명하지 않고 물가·성장·기간 프리미엄·발행 수급을 분리한다.
- 예측 적중률은 성숙·채점 표본이 없으면 `N/A`다.

## 출력

- 기준일 수익률곡선과 변화 분해.
- 국면 판정, 시나리오 확률, 핵심 촉매와 반대 근거.
- 만기·듀레이션·신용 버킷별 순풍과 역풍.
- 확인일, 무효화 조건, 실행 허용 범위.
