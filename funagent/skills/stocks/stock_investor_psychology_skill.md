# 스킬: 투자자 심리 해석

## 목적

주식 정보를 시장 참여자의 기대, 포지션, 가격 반응, 수급 지속성으로 번역해 전략 판단에 사용한다.

## 핵심 모델

```text
정보 -> 기대치 -> 포지션 -> 가격 반응 -> 수급 지속성
```

이 모델을 통과하지 못한 정보는 매수/매도 근거로 쓰지 않는다.

## 절차

1. 정보가 새 정보인지, 이미 알려진 정보인지 확인한다.
2. 시장 기대 대비 좋은지 나쁜지 판단한다.
3. 정보 발표 전 주가와 거래량이 선반영을 암시하는지 본다.
4. 발표 후 가격 반응을 본다.
5. 발표 후 거래량이 매집인지 분산인지 판단한다.
6. 기관, 외국인, 개인, 프로그램 수급의 지속성을 본다.
7. 환율이 외국인 수급, 수출입 마진, 위험선호에 순풍인지 역풍인지 분리한다.
8. FOMO, 공포, 본전 심리, 처분효과, 과잉확신, 제한된 주의 플래그를 부여한다.
9. 심리 해석 결과를 `진입`, `관찰`, `보류`, `회피` 중 하나로 분류한다.

## 해석 공식

- 호재 + 약한 주가 = 선반영, 기대 이하, 매도 대기 가능성.
- 악재 + 안 밀림 = 악재 선반영, 투매 종료 가능성.
- 급등 + 개인 과열 + 윗꼬리 = 분산 의심.
- 조정 + 거래량 감소 + 수급 유지 = 건강한 눌림 가능성.
- 돌파 + 거래대금 증가 + 종가 유지 = 매물 흡수 가능성.
- 호재 + 기관/외국인 동반 매도 + VWAP 이탈 = 기대 선반영 또는 분산 의심.
- 환율 역풍 + 외국인 매도 지속 = 단기 매수 확률 하향.
- 환율 순풍 + 외국인 매수 전환 + 섹터 상대강도 = 단기 확률 우위 가능성.

## 공격적 단기 매매 해석

- 공격적 단기 매매는 기대치, 포지션, 가격 반응, 수급 지속성에 더해 기관/외국인/프로그램 수급과 환율 상태를 별도 근거로 기록한다.
- 수급과 환율이 가격 반응과 충돌하면 심리 해석은 `관찰` 또는 `보류`로 낮춘다.
- 개인 과열이 강하고 기관/외국인 확인이 없으면 급등 추격이 아니라 분산 위험으로 먼저 해석한다.

## 출력

- 기대치 상태: `under_expected`, `in_line`, `above_expected`, `unknown`.
- 포지션 상태: `under_owned`, `crowded`, `overcrowded`, `unknown`.
- 가격 반응: `strong_close`, `weak_close`, `tail_risk`, `recovery`, `no_response`.
- 수급 지속성: `weak`, `mixed`, `improving`, `persistent`.
- 기관/외국인/프로그램 수급: `aligned`, `mixed`, `conflicting`, `unknown`.
- 환율 상태: `tailwind`, `neutral`, `headwind`, `unknown`.
- 심리 플래그.
- 전략 판정.
