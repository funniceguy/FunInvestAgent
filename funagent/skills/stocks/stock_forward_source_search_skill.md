# 스킬: 차주/미래 전망 신뢰 출처 검색

## 목적

차주 또는 미래 전망을 기반으로 주식 전략을 세울 때 세계정세, 경제상황, 정책 일정, 환율/금리/원자재, 개인·외국인·기관 심리를 신뢰 가능한 출처에서 검색하고, 전략에 쓸 수 있는 근거 후보만 남긴다.

## 적용 조건

- 사용자가 차주, 다음 주, 향후, 전망, 예측, 시나리오, 글로벌 정세, 경제상황 기반 전략을 요청한 경우.
- 단타, 공격적 단기, 가치투자, 레버리지 전략을 미래 전망으로 연결하려는 경우.
- 최신 출처 확인 없이 과거 지식이나 리포트 관성으로 전망을 작성하려는 경우.

## 출처 우선순위

1. 공식/1차 출처: 중앙은행, 통계기관, 거래소, 감독기관, 공시, 국제기구, 정부 제재/정책 발표.
2. 시장 데이터 출처: 거래소 수급, CFTC 포지션, 금리선물/옵션 기대, ETF 자금 흐름, 환율·금리·원자재 데이터.
3. 신뢰 보조 출처: 주요 통신사, 대형 리서치, 산업 전문기관. 공식 출처가 지연되거나 정성 사건을 설명할 때만 보조로 쓴다.
4. 금지 단독 출처: 블로그, 커뮤니티, 유튜브, 종목방, 제목만 있는 뉴스, 출처 없는 캡처.

## 필수 검색 영역

| 영역 | 최소 확인 |
|---|---|
| 세계정세 | 전쟁, 제재, 무역, 선거, 외교 갈등, 공급망 리스크 |
| 경제상황 | 금리, 물가, 고용, 소비, GDP, PMI, 경기 서프라이즈 |
| 유동성/환율 | 달러, 원/달러, 국채금리, 신용스프레드, 중앙은행 일정 |
| 원자재 | 유가, 가스, 구리, 금, 곡물 중 관련 섹터 민감 자산 |
| 투자자 심리 | 개인 과열/공포, 기관·외국인 수급, 선물 포지션, ETF 자금 흐름 |
| 이벤트 캘린더 | 차주 경제지표, FOMC/ECB/BOJ/BOK, 실적, 옵션만기, 휴장 |

## 우선 source_id

- 금리/중앙은행: `fed_fomc_calendar`, `cme_fedwatch_tool`, `ecb_governing_council_calendar`, `boj_monetary_policy_meetings`, `bok_ecos_api`.
- 경제지표 일정: `bls_release_calendar`, `bea_release_schedule`, `fred_api`.
- 환율/원자재/포지션: `fred_api`, `eia_weekly_petroleum_status`, `cftc_commitments_of_traders`.
- 세계정세/제재: `us_treasury_ofac`, 공식 정부·국제기구 발표.
- 개인/기관 심리: `aaii_sentiment_survey`, `krx_data_marketplace`, `kcmi_behavioral_bias`, `kcmi_investor_type_trading`.
- 장기 글로벌 전망: `imf_world_economic_outlook`, `worldbank_global_economic_prospects`.

## 검색 절차

1. 전망 기간을 `next_week`, `1_month`, `quarter`, `6_12_month`, `unknown`으로 분류한다.
2. 시장 범위를 한국, 미국, 글로벌, 섹터, 특정 종목으로 나눈다.
3. `funmemory/sources/source_registry.yaml`에서 공식/1차 출처 ID를 먼저 고른다.
4. 검색어를 `시장 + 기간 + 이벤트 + 데이터명` 형태로 만든다.
5. 각 필수 검색 영역에서 최소 1개 이상 출처를 확보한다. 공격적 단기 전략은 수급·환율·이벤트 캘린더를 반드시 포함한다.
6. 출처별 `published_at`, `data_as_of`, `source_checked_at`, `source_tier`를 기록한다.
7. 공식 출처와 보조 출처가 충돌하면 공식 출처를 우선하고, 충돌 사실을 리포트에 남긴다.

## 최신성 기준

- 가격, 금리, 환율, 원자재, 수급: 당일 또는 직전 거래일.
- 경제지표와 포지션 데이터: 최신 발표치와 다음 발표 일정.
- 정책·제재·세계정세: 보고서 작성 당일 검색.
- 장기 전망 보고서: 최신 발간본을 쓰되, 차주 전략에는 당일 시장 가격 반응으로 재검증한다.

## 출력

- 검색 범위와 전망 기간.
- 출처 후보 목록: source_id, URL, source_tier, published_at, data_as_of, source_checked_at.
- 필수 영역별 커버리지.
- 공식 출처와 보조 출처의 충돌 여부.
- 누락 출처와 결론 제한 사유.
- 다음 단계로 넘길 발췌 대상 목록.
