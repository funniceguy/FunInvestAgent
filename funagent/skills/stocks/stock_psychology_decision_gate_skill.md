# 스킬: 투자자 심리 기반 투자판단 게이트

## 목적

전략 수립 전에 투자자 심리 해석이 실제 행동을 허용할 만큼 강한지 판단한다. 이 게이트를 통과하지 못한 종목은 단타와 가치투자 모두 `관찰` 또는 `보류`로 낮춘다.

## 필수 질문 10개

각 항목은 0-10점으로 평가한다.

| 항목 | 질문 | 통과 기준 |
|---|---|---|
| 정보 신선도 | 새 정보인가, 이미 알려졌는가 | 7점 이상 |
| 실적 연결성 | 숫자와 현금흐름으로 연결되는가 | 7점 이상 |
| 기대 대비 | 시장 기대보다 좋은가 | 7점 이상 |
| 선반영 위험 | 정보 전 주가가 과열되지 않았는가 | 7점 이상 |
| 가격 반응 | 뉴스 이후 종가가 강한가 | 7점 이상 |
| 거래량 해석 | 매집/흡수에 가까운가 | 7점 이상 |
| 수급 지속성 | 기관/외국인/프로그램 수급이 이어지는가 | 7점 이상 |
| 개인 과열 | 개인 추격과 커뮤니티 과열이 낮은가 | 7점 이상 |
| 매물대 부담 | 본전 매물대를 흡수할 거래대금이 있는가 | 7점 이상 |
| 선발대 여부 | 지금 진입이 후발 추격이 아닌가 | 7점 이상 |

## 판정

- 평균 8.0 이상, 7점 미만 항목 0개: `actionable`.
- 평균 7.0 이상, 7점 미만 항목 2개 이하: `conditional_watch`.
- 평균 6.0 이상: `watch_only`.
- 평균 6.0 미만 또는 중대 위험: `avoid`.

## 중대 위험

- 장전 뉴스와 장초반 갭상승만 보고 추격.
- 급등, 개인 과열, 긴 윗꼬리가 동시에 발생.
- 호재에도 종가가 약하고 기관/외국인이 매도.
- 공시 원문 없이 뉴스 제목만으로 판단.
- 손절 또는 논리 훼손 조건이 없음.

## 출력

- 10개 항목 점수.
- 평균 점수.
- 중대 위험.
- 최종 판정.
- 단타/가치투자별 허용 행동.
