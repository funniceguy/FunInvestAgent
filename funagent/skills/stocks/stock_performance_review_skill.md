# 스킬: 주식 전략 성과 검토와 재수정

## 목적

수립한 단타 및 가치투자 전략이 현실적으로 작동했는지 검토하고, 성공/실패 원인을 분리해 다음 전략을 수정한다.

## 검토 주기

- 단타: 거래 종료 직후, 주간 합산, 월간 손실 한도 검토.
- 가치투자: 실적 발표, 주요 공시, 분기, 반기, 투자 논리 훼손 이벤트.
- 포트폴리오: 월간 리밸런싱 점검, 분기 전략 재평가.
- 장기 총수익·인컴: 월간 현금흐름, 분기 분배 재원·NAV, 반기 벤치마크·리밸런싱, 연간 세후 목표 재설계.

## 검토 지표

- 절대 수익률.
- 벤치마크 대비 수익률.
- 최대낙폭.
- 계획 준수율.
- 손절 준수율.
- 평균 손익비.
- 단타 승률과 기대값.
- 공격적 단기 매매 확률 게이트의 기대치, 수급, 환율, 가격 반응, 손익비 판단 적중 여부.
- 가치투자 논리 유지 여부.
- 세금과 비용 반영 후 성과.
- 원래 차주/미래 전망 시나리오와 실제 세계정세·경제지표·수급 반응의 일치도.
- 금주 예측 복기·평점 보정 루프가 이전 보고서를 충분히 읽고 성공/실패/판정 불가를 분리했는지 여부.
- 보정된 Verify 최소 점수와 필수 데이터 조건이 실제로 적용되었는지 여부.
- 반복 실패 원인이 다음 전략에서 줄었는지 여부.
- 원래 예측 보정·적중 추적의 상승/혼조/하락 확률, 예상 범위, 수급 방향, 환율 방향이 실제 결과와 얼마나 일치했는지.
- prediction_id별 방향 적중, 확률 오차, 범위 오차, 수급 오차, 환율 오차.
- 원래 전망 실패 사후분석에서 만든 재발 방지 규칙 준수 여부.
- 금리, 환율, 정부정책, 세계정세, 원자재, 유동성, 섹터 선행 지표의 종목별 민감도 판단 적중 여부.
- 공식 컨센서스, 스트리트 기대치, 주가 선반영 기대, 발표 후 가격 반응을 분리했는지 여부.
- 외국인, 기관, 프로그램, 개인 중 실제 가격 주도권을 잡은 주체를 맞췄는지 여부.
- NXT, ADR, 프리마켓, 해외 동종주 같은 선행 가격을 정규장 VWAP·수급으로 재검증했는지 여부.
- 매매 효율 점수와 실제 손익비, 유동성, 이벤트 타이밍, 상관 리스크의 일치도.
- 원래 심리 가정과 실제 가격/수급 반응의 일치도.
- FOMO, 공포, 본전 매물대, 처분효과, 과잉확신을 놓쳤는지 여부.
- 국내·미국, 장중·다음날·주간, 급등·순환·방어 국면별 방향 적중률과 Brier score를 분리했는지 여부.
- matured_unscored, legacy_unscorable, 중복 prediction_id, 결과 기준 사후 변경 여부.
- 예측 적중, 주문 체결·슬리피지, 손절·비중 준수를 각각 평가했는지 여부.
- 표시 호가 불균형이 실제 체결·가격 반응으로 확인됐는지, 매도 오버행·숨은 공급을 놓쳤는지 여부.
- 급등주·순환주·구조적 주도주 클래스와 사이클 단계가 실제로 어떻게 전이됐는지 여부.
- 장기 목표수익률의 복리 환산과 실제 세후 총수익·최대 낙폭·회복 기간의 차이.
- 커버드콜·인컴 상품의 분배율이 아니라 분배 재투자 총수익, 주당 NAV, 원금환급, 기초자산 대비 기회비용.
- 정상화 EPS·FCF, ROIC, 부채·희석, 배당 커버리지 가정이 실제로 유지됐는지 여부.
- 자본보전 게이트의 계획 손실예산과 실제 최대 손실, 손절 체결 갭, 상관 스트레스, 회복수익률·기간의 차이.
- 원래 파킹·예금·적금·채권 대체재 판정과 실제 적용금리·세후수령액·우대달성·보호 합산의 차이.
- 신규 주식의 실제 비용 후 수익이 같은 기간 현금성 대체수익과 계획 위험 완충폭을 넘었는지 여부.

## 실패 분류

| 분류 | 의미 | 수정 방향 |
|---|---|---|
| 데이터 실패 | 잘못되거나 오래된 데이터 사용 | 출처와 기준일 강화 |
| 분석 실패 | 기업 또는 국면 판단 오류 | 지표와 반대 시나리오 보강 |
| 실행 실패 | 손절, 비중, 주문 규칙 미준수 | 포지션 크기 축소, 자동 기록 |
| 리스크 실패 | 예외 상황을 과소평가 | 손실 한도와 중단 조건 강화 |
| 운의 영역 | 계획은 맞았으나 확률적 손실 | 전략 유지 또는 샘플 확대 |
| 예측 복기 실패 | 금주 보고서와 리뷰를 읽지 않거나 성공/실패를 분리하지 못함 | 금주 예측 복기·평점 보정 적합성을 필수 게이트로 승격 |
| 심리 해석 실패 | 정보는 맞았지만 기대치, 포지션, 가격 반응, 수급 지속성을 잘못 읽음 | 심리 게이트 기준 강화 |
| 확률 게이트 실패 | 기관/외국인 수급, 환율, VWAP, 거래대금, 손익비 정렬을 잘못 판단 | 공격적 단기 게이트 기준 강화 또는 watch_only로 하향 |
| 예측 보정 실패 | 방향은 말했지만 확률, 근거 수치, 검증 시각, 오차 계산이 부정확했음 | 예측 보정·적중 추적 가중치와 데이터 필수 조건 강화 |
| 전망 근거 실패 | 검색 출처, 근거 카드, 시나리오, 세계정세·경제·수급 전달 경로를 잘못 해석 | 검색 커버리지와 시나리오 무효화 조건 강화 |
| 실패 회고 실패 | 이전 무효화 조건과 최초 경고 신호를 다음 전략에 반영하지 않음 | 전망 실패 사후분석을 필수 게이트로 승격 |
| 거시 민감도 실패 | 금리, 환율, 정책, 세계정세를 종목별로 다르게 적용하지 못함 | 거시 원인 민감도 매트릭스와 효율 점수 기준 강화 |
| 숨은 기대치 실패 | 공식 컨센서스는 이겼지만 주가에 반영된 기대치를 못 넘긴 이벤트를 호재로 오판 | 이벤트 전날 신규 금지, 발표 후 VWAP/종가 반응 확인 |
| 주도권 수급 실패 | 개인 순매수를 좋은 수급으로 보고 외국인·기관·프로그램 매도를 과소평가 | 주도권 수급 충돌 시 신규 진입 watch_only 이하 |
| 이벤트 분산 실패 | 실적·상장·수주 발표일이 선취매 세력의 매도 유동성으로 사용됨 | 이벤트 당일 추가매수보다 축소·재평가 우선 |
| 선행시장 오판 | NXT, ADR, 프리마켓 강세를 정규장 확정 신호로 사용 | 선행가격은 기준선으로만 쓰고 KRX/정규장 실패 시 철수 |
| 전수 감사 실패 | 성공 사례만 선택하거나 미채점 예측을 누락 | 전체 원장 재구축, 시장·기간·국면별 분리 |
| 잠재 물량 실패 | 호가벽을 실제 수요·공급으로 오판하거나 숨은 공급 누락 | depth 지속성·실제 체결·가격 반응·공시 공급 재검증 |
| 사이클 분류 실패 | 급등 말기나 순환 말기를 초기 주도 구간으로 오판 | 클래스 확률·신뢰도·단계 전환 트리거 강화 |
| 장기 목표 실패 | 월·연 복리 또는 납입·인출 가정을 과도하게 설정 | 목표 기간·납입·인출·현금 버퍼 재설계 |
| 분배금 착시 실패 | 높은 분배율을 총수익으로 오인하거나 원금환급·NAV 훼손을 누락 | 분배 재원·NAV·기초자산 총수익 게이트 강화 |
| 장기 가치함정 실패 | 낮은 PER/PBR이 정상화 이익 감소·낮은 ROE를 반영 | 정상화 FCF·ROIC·희석·멀티플 의존도 기준 강화 |
| 자본보전 실패 | 손절가만 믿고 갭·슬리피지·상관 급등을 과소평가 | 스트레스 손실 기준 비중 축소와 손실예산 강화 |
| 회복 오판 | 구조적 훼손을 일시적 하락으로 보고 물타기·대기 | 회복 재원 3축 검증과 structural_impairment 차단 |
| 현금성 대체재 오판 | 최고금리 착시·보호 합산·중도해지·채권 YTW 비교를 잘못해 대기 또는 진입 결정을 왜곡 | 실제 세후 현금흐름과 안전 대체수익 기준 재계산 |

## 출력

- 전략별 성공/실패 판정.
- 실패 원인.
- 유지할 규칙.
- 폐기할 규칙.
- 수정된 다음 전략.
- 다음 검토일.
- 금주 예측 복기·평점 보정 수정 사항.
- 심리 게이트 수정 사항.
- 전망 실패 사후분석 수정 사항.
- 예측 보정·적중 추적 수정 사항.
- 거시 원인 민감도 매트릭스 수정 사항.
- 전망 검색·발췌·분석 수정 사항.
- 공격적 단기 매매 확률 게이트 수정 사항.
- 숨은 기대치 보정 규칙.
- 주도권 수급 보정 규칙.
- NXT/ADR/프리마켓 선행가격 보정 규칙.
- 국내·미국 전수 성과 감사와 표본 등급.
- 잠재 매수·매도 물량 및 오버행 판정 수정 사항.
- 급등주·순환주 사이클 분류 수정 사항.
- 장기 목표 현실성, 밸류에이션·내구성, 분배·커버드콜 품질, 장기 포트폴리오 수정 사항.
- 자본보전·손실회복 점수, 계획·실제 낙폭, 회복 품질과 손실예산 수정 사항.
- 현금성·예적금·채권 대체재의 실제 수익, 보호·조건 변화와 다음 안전 대안 규칙.
