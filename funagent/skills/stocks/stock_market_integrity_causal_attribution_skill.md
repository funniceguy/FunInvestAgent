# 스킬: 시장충격 인과·시장무결성 검증 게이트

## 목적

폭락·급등·세력·정치 압박·시세조종·전쟁·레버리지 의혹이 있는 시장에서 가격 움직임의 원인을 `구조적 취약성 -> 방아쇠 -> 증폭기 -> 결과`로 분해하고, 합법적인 대형 수급·기계적 매도와 불법 시세조종 주장을 증거 수준에 따라 분리한다. 이 스킬은 범인을 추정하는 도구가 아니라 전략에 사용할 수 있는 확인된 인과와 아직 입증되지 않은 가설을 구분하는 상위 게이트다.

## 적용 조건

- 사용자가 `세력`, `장난질`, `조작`, `정치권 압박`, `정부가 누른다`, `전쟁 때문`, `시장 박살`, `비정상 폭락`을 언급한 경우.
- 사이드카·서킷브레이커, 지수 또는 대형주 급락, 단일종목 ±2배 ETF·ETN, 프로그램 대량매도, ADR/원주 가격 괴리가 있는 경우.
- 폭락 뒤 반등 전략, 물타기, 레버리지 신규 진입, 시장 전체 자금 이탈 여부를 판단하는 경우.
- 이전 전략이 거시 뉴스, 기관 합계, 외국인 수급 또는 레버리지 상품을 원인으로 잘못 단정한 경우.

## 선행 게이트

1. `stock_data_collection_skill.md`: 동일 기준시각의 가격·수급·거시·선행시장 자료를 수집한다.
2. `stock_mechanical_flow_risk_gate_skill.md`: 선물·프로그램·기관 세부·레버리지·사이드카를 판정한다.
3. `stock_macro_causal_factor_matrix_skill.md`: 전쟁·유가·금리·환율·정책의 전달 경로를 종목별로 계산한다.
4. 신규 주문이면 `stock_supply_overhang_order_flow_skill.md`의 주문·체결·공급 자료를 추가한다.

선행 게이트가 `unknown`이거나 핵심 수급 기준시각이 다르면 인과 결론은 `conditional_attribution` 이하로 제한한다.

## 핵심 원칙

- 큰 폭의 가격 변화는 조작의 증거가 아니다. `시장 영향이 큼`, `기계적 매도가 존재함`, `불법 공모가 입증됨`을 서로 다른 판정으로 기록한다.
- 원인은 최소 `구조적 취약성`, `방아쇠`, `증폭기`, `결과`, `반대 가설`로 분해한다. 같은 사실을 여러 원인 금액에 중복 합산하지 않는다.
- 기관 합계는 금융투자, 투신, 사모, 보험, 은행, 연기금, 기타로 가능한 범위에서 분해한다. 정치·연기금 가설은 연기금 실제 기여액을 투신·사모·프로그램과 비교하기 전 사용하지 않는다.
- 차익과 비차익 프로그램을 분리한다. 비차익 매도는 바스켓·ETF·인덱스·환매·모델 매도의 정황이지만, 그 자체가 불법 조작 또는 특정 레버리지 상품의 정확한 매도액을 증명하지 않는다.
- 개인 순매수가 크고 가격이 하락하면 `retail_absorption_risk`다. 자금이 남아 있다는 뜻과 가격이 바닥이라는 뜻을 동일시하지 않는다.
- 시장 전체 자금 이탈과 섹터 회전을 구분하려면 지수별 낙폭, 상승/하락 종목수, 섹터 상대강도, 개인·외국인·기관·프로그램 수급을 함께 본다.
- 전쟁·유가·정책 뉴스는 글로벌 동종시장과 비교한 반사실 검증을 통과해야 한다. 해외 선행시장보다 국내 낙폭이 현저히 크면 외부 충격을 단독 원인으로 쓰지 않는다.
- ADR/원주 가격 괴리는 가격발견 신호일 수 있지만 전환비율, 전환 가능성, 결제, 세금, 환율, 차입·공매도, 추가 발행을 확인하기 전 무위험 차익 또는 즉시 매수 신호로 쓰지 않는다.
- 공식 조사 발표를 찾지 못한 것은 조작이 없다는 증명이 아니다. `official_finding_not_found`와 `manipulation_refuted`를 혼동하지 않는다.

## 필수 입력

| 영역 | 필수 데이터 | 검증 목적 |
|---|---|---|
| 가격·집중 | 지수·대형주·섹터 낙폭, 거래대금, 시가총액 집중, 상승/하락 종목수 | 시장 전체 충격과 집중 충격 분리 |
| 주체 수급 | 개인, 외국인, 기관 합계와 금융투자·투신·사모·연기금 등 세부 | 실제 매도 주도권과 정치 가설 비교 |
| 프로그램·파생 | 차익/비차익, 선물 주체별 수급, 베이시스, 사이드카·서킷브레이커 | 기계적 증폭과 유동성 고갈 확인 |
| 레버리지 상품 | +2x/-2x 상품 구조, AUM, 거래대금, 설정·환매, NAV/IOPV, 괴리, 헤지수단 가능 시 | 일일 재설정과 상품별 귀속 한계 |
| 사건·기대치 | 실적 추정 변화, 상장·ADR, 정책, 공시, 이벤트 전후 가격 반응 | 방아쇠와 선반영 분리 |
| 반사실 시장 | 해외 동종주, 섹터 ETF, 선물, 다른 국내 지수·섹터 | 외부 충격의 단독 설명력 검증 |
| 거시 | 유가, 금리, 환율, 달러, 지정학 타임라인 | 시장 전달 경로 확인 |
| 무결성 자료 | 감독당국·거래소 공식 발표, 계좌·주문·취소·현물-파생 연계 자료 가능 시 | 조작 혐의의 증거등급 판정 |
| ADR/원주 | 전환비율, 환율, 원주·ADR 가격, 전환·결제·세금·차입 제약 | 가격 괴리의 실행 가능성 판정 |

필수 영역 중 가격·주체 수급·프로그램·반사실 시장·공식 발표의 5개 축 가운데 2개 이상이 없으면 `causal_attribution_unresolved`다.

## 인과 계층

| 계층 | 정의 | 예시 |
|---|---|---|
| `structural_vulnerability` | 충격 전부터 낙폭을 키울 수 있던 조건 | 지수 집중, 급등·과밀, 신용, 단일종목 레버리지 AUM |
| `trigger` | 최초 위험 축소를 촉발한 새 정보 | 이익 기대 하향, 상장 후 뉴스 매도, 전쟁 재격화 |
| `amplifier` | 최초 충격을 반복·확대하는 흐름 | 비차익 프로그램, ETF/스왑 헤지, 변동성 타깃, 강제청산 |
| `consequence` | 원인과 구분해야 할 결과 | 사이드카, 서킷브레이커, 개인 대량 저가매수, 변동성 확대 |
| `alternative_hypothesis` | 경쟁 설명과 반대 증거 | 정치 압박, 공모 조작, 전쟁 단독 원인, 전면 자금 이탈 |

각 원인에는 `evidence_level`, `confidence`, `source_ids`, `counterevidence`, `unknowns`를 붙인다. 신뢰도는 사실의 존재가 아니라 해당 원인이 관찰 결과를 설명하는 정도다.

## 레버리지 일일 재설정 검증

시작 순자산을 `V`, 목표 배수를 `L`, 기초자산 일간 수익률을 `r`로 단순화하면 종가 목표 노출 조정은 대략 `L × V × r × (L - 1)` 방향이다.

- `L=+2`: 상승 후 추가 매수, 하락 후 노출 축소 매도.
- `L=-2`: 상승 후 숏 축소를 위한 매수, 하락 후 숏 확대를 위한 매도.
- 따라서 +2배와 -2배 모두 큰 방향 움직임 뒤 운용·스왑 상대방 헤지가 같은 방향을 따라 변동성을 키울 수 있다.
- 인버스 급등 뒤 레버리지 급등이 순차적으로 나타나는 화면은 기초자산의 급락·반등과 일일 배수 산술만으로도 발생하므로 조작 증거로 쓰지 않는다.
- 실제 주문은 현물·선물·스왑·장외 헤지와 설정·환매에 따라 달라진다. 상품별 설정·환매와 헤지 장부 없이 정확한 매도액, 주문 시각, 최종 주체를 특정하지 않는다.

## 정치·시세조종 가설 증거등급

| 상태 | 요구 증거 | 전략 사용 |
|---|---|---|
| `confirmed_official` | 감독당국·거래소·법원의 공식 확인 | 사실로 사용하되 조사와 확정판결을 구분 |
| `supported_investigative` | 공식 조사 개시와 계좌·주문 연계 정황 | 높은 위험 가설, 법적 확정으로 표현 금지 |
| `public_flow_only` | 대형 수급·프로그램·종가 충격만 확인 | 시장 영향은 인정, 불법 의도는 미입증 |
| `unproven` | 가격 움직임·소문·정치적 추정만 존재 | 전략 인과의 핵심 근거로 사용 금지 |
| `not_assessed` | 공식 자료 검색 또는 필수 데이터 미완료 | 인과 게이트 보류 |

불법 공모를 강하게 지지하는 자료는 연계 계좌 집중, 높은 주문취소·허수벽, 반복 종가 관여, 현물 매도 전 파생 이익 포지션, 경제적 이유 없는 가격무시 주문, 통신·지시·수익 귀속이다. 공개 주체별 일별 합계만으로는 이를 입증할 수 없다.

## 시장 자금 상태 판정

- `broad_exit`: 다수 지수·섹터가 동반 하락하고 외국인·기관·프로그램이 광범위하게 위험을 축소한다.
- `sector_rotation`: 지수 약세에도 상승 섹터·종목 폭이 유의미하고 매도 자금이 다른 섹터로 이동한다.
- `mixed_exit_rotation`: 외국인·기관의 실제 위험 축소와 일부 섹터 회전이 동시에 있다.
- `unknown`: 시장 폭·주체·섹터 데이터가 불충분하다.

## 점수화

각 항목은 0-10점이며 높을수록 인과 분석과 증거 규율이 완전하다.

| 항목 | 10점 기준 |
|---|---|
| 인과 계층 완결성 | 취약성·방아쇠·증폭기·결과·반대 가설을 분리 |
| 주체·프로그램 분해 | 기관 세부와 차익/비차익을 동일 기준시각으로 확인 |
| 반사실·교차시장 | 글로벌 동종주·다른 지수·섹터 폭으로 단독 원인 검증 |
| 레버리지 구조 | ±2배 재설정과 설정·환매·헤지 귀속 한계를 모두 반영 |
| ADR/원주 가격발견 | 전환비율·환율·실행 제약·괴리 축소 경로 확인 |
| 정치·조작 증거 규율 | 공식 확인·조사·공개수급·소문을 등급화 |
| 자금 이탈·회전 구분 | 시장 폭과 주체 수급으로 broad/rotation/mixed 판정 |
| 전략 연결 | 확률 상한, 신규 진입, 레버리지, 재검증 조건으로 연결 |
| 최신성·출처 | 당일 데이터와 공식/1차 출처, 기준시각 완비 |
| 불확실성 기록 | unknown, 반대 증거, 중복 집계 위험을 명시 |

판정:

- `causal_attribution_complete`: 96점 이상, 중대 차단 0개, 핵심 unknown 0개.
- `conditional_attribution`: 85-95점, 핵심 결론은 사용 가능하나 계좌·상품별 자료 등 한계 존재.
- `causal_attribution_unresolved`: 70-84점 또는 필수 축 2개 이상 누락.
- `causal_attribution_blocked`: 70점 미만 또는 중대 차단 조건 존재.

`conditional_attribution`은 조작 가설이 확인됐다는 뜻이 아니다. 공개자료 기반 시장 원인 분석은 계좌 자료가 없으면 높은 절차 점수여도 조작 상태를 `public_flow_only` 또는 `unproven`으로 유지할 수 있다.

## 전략 연결

- `causal_attribution_complete`: 다른 게이트가 clear일 때만 인과를 전략 가중치로 사용한다.
- `conditional_attribution`: 주방향 확률 상한 60%, 신규 레버리지 금지, 현물도 기계적 수급 caution 이상 확인 후 검토한다.
- `causal_attribution_unresolved`: 주방향 확률 상한 50%, 신규 진입·물타기·레버리지 금지, 관찰만 허용한다.
- `causal_attribution_blocked`: 방향 전략을 만들지 않고 데이터 수집과 위험 축소 조건만 제시한다.
- 조작 상태가 `unproven` 또는 `public_flow_only`이면 조작 추정을 매수·매도 근거로 쓰지 않는다.
- 정치·연기금 가설의 실제 기여액이 투신·사모·프로그램보다 현저히 작으면 정치 가설을 지배적 원인으로 쓰지 않는다.
- 전쟁·유가가 글로벌 선행시장보다 국내 낙폭을 충분히 설명하지 못하면 `trigger` 또는 `amplifier`로만 사용한다.

## 출력

- `market_integrity_causal_attribution_packet`
- 데이터 기준시각과 출처 등급.
- 구조적 취약성·방아쇠·증폭기·결과·반대 가설 표.
- 개인·외국인·기관 세부, 차익·비차익 프로그램과 중복 집계 주의.
- 레버리지 ±2배 재설정, 상품별 설정·환매·헤지 귀속 한계.
- 해외 동종주·다른 지수·섹터 폭·ADR/원주 반사실 비교.
- 시장 자금 상태: `broad_exit`, `sector_rotation`, `mixed_exit_rotation`, `unknown`.
- 정치·시세조종 상태: `confirmed_official`, `supported_investigative`, `public_flow_only`, `unproven`, `not_assessed`.
- 최종 판정과 전략 확률·비중·레버리지 상한.
- 조작 가설을 확인하거나 기각하기 위해 추가로 필요한 자료.

## 중대 차단 조건

- 가격이 크게 움직였다는 이유만으로 정치 압박·세력·불법 조작을 사실로 단정한다.
- 기관 합계만 보고 연기금·정치 가설을 지배적 원인으로 쓰거나 투신·사모·프로그램과 비교하지 않는다.
- 비차익 프로그램 매도를 전부 레버리지 ETF 매도 또는 불법 알고리즘 주문으로 귀속한다.
- +2배 또는 -2배 한쪽만 설명하고 양방향 일일 재설정의 동방향 헤지 가능성을 누락한다.
- 상품별 설정·환매·스왑·선물 헤지 자료 없이 레버리지 상품의 정확한 충격액을 단정한다.
- 전쟁·유가·정책 뉴스를 해외 동종시장과 비교하지 않고 단독 원인으로 확정한다.
- ADR/원주 괴리를 전환·결제·세금·환율·차입 제약 없이 무위험 차익 또는 즉시 매수 신호로 사용한다.
- 개인 순매수와 일부 상승 종목만 보고 전체 자금 이탈이 없다고 단정하거나, 외국인·기관 순매도만 보고 섹터 회전을 부정한다.
- 공식 조사 발표를 찾지 못했다는 이유로 조작이 없다고 확정하거나, 공식 조사 개시를 유죄 확정처럼 표현한다.
- 인과 게이트가 unresolved/blocked인데 신규 진입·물타기·레버리지 전략을 승인한다.
