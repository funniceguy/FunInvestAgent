# 스킬: 전망 실패 사후분석

## 목적

이전 전략이 현실 가격 반응을 따라가지 못했거나 사용자가 `한발 늦다`, `전망 실패`, `박살`, `왜 틀렸나`, `전략을 개선`이라고 요청한 경우, 기존 전망의 가정과 실제 결과를 엄격하게 비교해 다음 전략의 규칙을 강화한다. 이 스킬은 변명용 회고가 아니라, 실패 원인을 재발 방지 규칙과 신규 게이트로 바꾸는 단계다.

## 적용 조건

- 사용자가 이전 전략의 실패, 지연, 급락, 손실, 진입 실패, 매도 실패, 뒤늦은 대응을 언급한 경우.
- 같은 섹터나 종목에 대해 새 전략을 요청하지만 직전 전략의 무효화 조건이 이미 깨진 경우.
- 호재에도 가격이 하락하거나 악재에도 가격이 버티는 등 전망과 가격 반응이 충돌한 경우.
- 차주/미래 전망 전략을 재작성하기 전.
- 금일/내일/차주 전략을 만들기 전 금주 예측 복기에서 실패 예측 또는 부분 성공 예측이 발견된 경우.
- 공격적 단기 매매 게이트를 통과시키기 전.

## 핵심 원칙

- 전략 실패는 `예측이 틀렸다`로 끝내지 않고, 어떤 입력·가정·게이트·실행 규칙이 부족했는지 분해한다.
- 좋은 산업, 좋은 기업, 좋은 뉴스, 좋은 진입가는 각각 별도 판정한다.
- 가격 반응이 전망과 충돌하면 가격 반응을 우선하고, 전망은 `조건부`, `관찰`, `중단` 중 하나로 낮춘다.
- 직전 전략의 무효화 조건이 깨졌으면 같은 논리로 신규 진입하거나 물타기하지 않는다.
- 실패 원인이 확인되기 전에는 신규 전략 결론을 `진행 가능`으로 낼 수 없다.
- 금주 복기에서 성공 예측이 함께 발견되면 성공 요인도 기록하되, 실패 원인을 덮는 근거로 쓰지 않는다.
- `stock_forecast_learning_loop_skill.md`의 반복 실패 원인은 사후분석의 우선 조사 항목으로 삼고, 현재 전략의 차단 조건이나 평점 강화안으로 연결한다.

## 필수 입력

- 이전 전략 파일 또는 이전 전략 요약: 핵심 가정, 후보, 진입 조건, 손절/무효화 조건, 시나리오.
- 실제 결과: 기준 시각, 현재가 또는 종가, 수익률, 지수·섹터 대비 상대강도, 거래대금, 호가/스프레드.
- 이벤트 타임라인: 공시, 실적, 가이던스, 정책, 금리, 환율, 세계정세, 섹터 뉴스, 애널리스트 의견, 실적 일정.
- 투자자 반응: 개인, 외국인, 기관, 프로그램, ETF/선물/옵션 흐름을 가능한 범위에서 분리.
- 누락 데이터: 실시간 VWAP, 기관·외국인·프로그램 수급, 환율, 금리, ETF NAV/IOPV, 브로커 주문 가능 시간 등.

## 실패 원인 분류

각 실패 원인을 0-10점으로 평가한다. 7점 이상은 다음 전략에서 강제 보완 규칙으로 승격한다.

| 실패 원인 | 질문 | 보완 방향 |
|---|---|---|
| 데이터 최신성 실패 | 가격, 호가, 수급, 금리, 환율이 당일 기준이었는가 | 최신 데이터 없으면 신규 진입 금지 |
| 기대치 해석 실패 | 뉴스가 시장 기대보다 충분히 좋았는가 | `호재 매도`면 기대 과포화로 분류 |
| 숨은 기대치 실패 | 공식 컨센서스는 이겼지만 주가에 반영된 스트리트 기대치를 못 넘겼는가 | 이벤트 전에는 `공식 컨센서스`, `주가 선반영 기대`, `가격 반응 임계값`을 분리 |
| 선반영·포지션 혼잡 실패 | 이미 급등하거나 거래가 과열됐는가 | 선반영 점수 높으면 추격 금지 |
| 이벤트 분산 실패 | 실적·상장·수주·정책 발표일이 선취매 세력의 매도 유동성으로 쓰였는가 | 이벤트 당일 추격 금지, 반등 시 축소/재평가 우선 |
| 선행 시장 누락 | 아시아, 유럽, 프리마켓, ADR, ETF가 먼저 경고했는가 | 선행 지표를 최상위 게이트로 승격 |
| NXT 선행가격 오판 | NXT 프리마켓 강세를 KRX 정규장 수급 확정으로 오해했는가 | NXT는 파일럿이 아니라 09:30 실패 시 철수 기준으로 사용 |
| 금리·환율 누락 | 성장주·수출주·외국인 수급에 금리/환율이 역풍이었는가 | 민감 종목은 금리/환율 확인 전 보류 |
| 정부·정책 누락 | 규제, 보조금, 제재, 관세, 수출통제가 실적·멀티플에 영향이 있었는가 | 정책 민감도 높은 종목은 정책 게이트 추가 |
| 세계정세 누락 | 전쟁, 제재, 외교 갈등, 공급망 충격이 반영됐는가 | 전달 경로 없으면 뉴스 단독 사용 금지 |
| 주도권 수급 실패 | 외국인·기관·프로그램 중 2개 이상이 반대 방향인데 개인 매수를 좋은 수급으로 봤는가 | 주도권 수급 충돌 시 신규 진입 `watch_only` 이하 |
| 수급·ETF·프로그램 누락 | 개별 호재보다 ETF/섹터 매도가 강했는가 | 섹터 ETF가 약하면 개별주 신규 매수 차단 |
| 기계적 수급 오판 | 외국인 현물보다 기관 선물, 프로그램, ETF/ETN, 단일종목 레버리지, LP 헤지가 가격을 주도했는가 | `stock_mechanical_flow_risk_gate_skill.md`를 상위 게이트로 적용 |
| 인과 계층 오판 | 구조적 취약성·방아쇠·증폭기·결과를 섞어 단일 원인으로 설명했는가 | `stock_market_integrity_causal_attribution_skill.md`로 계층 재분류 |
| 조작·정치 가설 과잉 | 공개 수급만으로 불법 공모나 정치 의도를 단정했는가 | 공식 조사·계좌·주문 증거 등급이 없으면 `unproven` 유지 |
| 레버리지 귀속 과잉 | 프로그램 매도 전부를 특정 ±2배 상품에 귀속했는가 | 설정·환매·스왑·선물 헤지 없이는 정확액 unknown |
| ADR/원주 괴리 오판 | 전환·결제·세금·환율·차입 제약 없이 괴리를 무위험 차익으로 봤는가 | 가격발견과 실행 가능한 차익을 분리 |
| 자금 이탈·회전 오판 | 개인 저가매수나 일부 상승주만으로 전체 자금 상태를 단정했는가 | 시장 폭·섹터 상대강도·주체 수급으로 broad/rotation/mixed 판정 |
| 시간대 판단 실패 | 오픈, 점심, 종가, FOMC/실적 전후 시간대가 반영됐는가 | 시간대별 행동 제한 강화 |
| 상관·집중 리스크 실패 | 같은 테마 포지션을 여러 종목으로 중복 보유했는가 | 섹터 상관 한도와 후보 수 제한 |
| 심리·행동 실패 | FOMO, 본전 심리, 손실 복구 물타기가 있었는가 | 신규 진입 대신 축소·관찰 우선 |
| 무효화 실행 실패 | 깨진 손절/무효화 조건을 즉시 실행했는가 | 무효화 이탈 시 자동 결론 하향 |
| 잠재 물량 오판 | 표시 호가를 실제 수요·공급으로 오해하거나 숨은·공시 공급을 놓쳤는가 | 복수 depth·체결·AVWAP·공급 캘린더 게이트 추가 |
| 급등·순환 사이클 오판 | 급등/순환 말기나 분산을 초기 주도 구간으로 봤는가 | 클래스 확률·신뢰도·단계 전환 조건 강화 |

## 실패 심각도

| 등급 | 조건 | 다음 전략 제한 |
|---|---|---|
| minor | 방향은 맞았으나 진입 위치 또는 시간대가 나쁨 | 비중 축소, 진입 조건 강화 |
| moderate | 핵심 조건 1-2개가 깨졌는데 대응이 늦음 | 신규 진입 `조건부 검토` 이하 |
| severe | 시장 국면, 섹터 방향, 기대치 해석이 모두 틀림 | 신규 진입 `보류` 또는 `중단 권고` |
| critical | 손절·무효화 조건이 깨졌는데 물타기 또는 확대를 유도함 | 같은 논리 사용 금지, 전략 재작성 |

## 사후분석 절차

1. 이전 전략의 핵심 가정을 3-7개로 압축한다.
2. 실제 가격 반응과 상대강도를 같은 기준 시각으로 기록한다.
3. `예상 시나리오`, `실제 시나리오`, `차이`, `처음 확인 가능했던 신호`를 표로 만든다.
4. 실패 원인을 위 분류표로 점수화한다.
5. 공식 컨센서스, 스트리트 기대치, 주가 선반영 기대, 발표 후 가격 반응을 분리해 `기대치 갭`을 기록한다.
6. 가장 먼저 감지 가능했던 선행 신호를 찾는다.
7. 선행 신호를 다음 전략의 `차단 조건`, `비중 축소 조건`, `관찰 조건`, `재진입 조건` 중 하나로 바꾼다.
8. 외국인, 기관, 프로그램, 개인 중 누가 가격 주도권을 잡았는지 `주도권 수급`으로 기록한다.
9. 선물, 프로그램, ETF/ETN, 단일종목 레버리지, 사이드카, LP 헤지 중 가격을 강제로 움직인 기계적 수급 체인을 별도로 기록한다.
10. 외국인 현물 신호가 실제 가격 방향과 충돌했으면 `foreign_cash_false_positive` 여부를 판정한다.
11. 사용자의 행동 리스크를 따로 기록한다.
12. 다음 전략에서 사용할 수정 규칙과 제거할 낡은 가정을 명시한다.
13. 성공 예측에서 유지할 규칙과 실패 예측에서 폐기할 규칙을 분리한다.
14. 반복 실패 원인이 있으면 관련 Verify 카테고리의 최소 점수 강화안을 함께 기록한다.
15. 호가·체결·잠재 공급과 급등·순환 사이클의 원래 판정이 실제 경로와 달랐는지 별도 채점한다.
16. 폭락·조작·정치·전쟁 가설이 있으면 인과 계층, 반사실 시장, 공식 조사 상태, ±2배 귀속 한계, 자금 이탈·회전 판정을 별도 채점한다.

## 출력

- `forecast_failure_postmortem_packet`
- 이전 전략 핵심 가정 요약.
- 실제 결과와 오차.
- 실패 원인 점수표.
- 최초 경고 신호와 놓친 시간.
- 재발 방지 규칙.
- 성공 예측에서 유지할 규칙과 실패 예측에서 제거할 가정.
- 관련 Verify 카테고리 평점 강화안.
- 다음 전략에 반드시 반영할 신규 게이트.
- 기계적 수급·프로그램 매도 게이트 판정과 외국인 현물 신호 유효성.
- 잠재 물량·오버행 및 급등·순환 사이클 오판 여부.
- 시장충격 인과·시장무결성 오판 여부와 조작 가설 증거등급.
- 결론 제한: `진행 가능`, `조건부 검토`, `보류`, `중단 권고`.

## 중대 차단 조건

- 이전 전략의 무효화 조건이 깨졌는데 실패 분석 없이 같은 방향 신규 진입을 제안한다.
- 호재에도 가격이 하락했는데 기대치 과포화 또는 수급 분산 가능성을 검토하지 않는다.
- 공식 컨센서스만 보고 호재로 판단하고, 주가 선반영 기대와 발표 후 VWAP/종가 반응을 비교하지 않는다.
- 외국인·기관·프로그램 중 2개 이상이 반대 방향인데 개인 순매수를 좋은 수급으로 분류한다.
- 외국인 현물 순매수만 보고 시장 안정으로 해석하고 기관 선물·프로그램·ETF/레버리지 리밸런싱을 확인하지 않는다.
- 단일종목 레버리지 ETF 또는 프로그램 매도 체인이 의심되는데 `stock_mechanical_flow_risk_gate_skill.md`로 신규 게이트를 만들지 않는다.
- 폭락 원인을 구조적 취약성·방아쇠·증폭기·결과로 분리하지 않거나, 공개 수급만으로 정치 압박·불법 조작을 단정한다.
- ADR/원주 괴리 또는 프로그램 총계를 실행 제약·상품별 귀속 한계 없이 확정 매매 근거로 사용한다.
- 이벤트 전날 또는 당일 급등 종목을 선취매 완료 가능성 없이 추격 후보로 분류한다.
- NXT 프리마켓 가격만으로 KRX 정규장 수급을 확정한다.
- 금리, 환율, 정부정책, 세계정세 중 해당 종목의 핵심 민감 요인을 확인하지 않는다.
- 섹터 ETF와 선행 시장이 약한데 개별주만 보고 저점매수를 제안한다.
- 실패 원인을 사용자 심리 또는 시장 탓으로만 돌리고 규칙 변경을 만들지 않는다.
