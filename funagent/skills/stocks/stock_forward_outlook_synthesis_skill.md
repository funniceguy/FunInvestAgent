# 스킬: 차주/미래 전망 시나리오 합성

## 목적

검색·발췌된 근거 카드를 세계정세, 경제상황, 환율/금리/원자재, 개인·외국인·기관 투자 심리로 합성해 차주 또는 미래 전략 방향을 만든다. 이 스킬은 방향성 시나리오를 만들고, 실제 상승/혼조/하락 확률과 예상 범위는 후속 `stock_predictive_calibration_skill.md`에서 보정한다.

## 적용 조건

- 차주 전략, 향후 전망, 미래 예측, 글로벌 매크로 기반 전략을 작성하는 경우.
- 단기 매매, 공격적 단기, 가치투자 후보 선정 전에 시장의 기본 방향과 리스크 시나리오를 정해야 하는 경우.
- 수급·환율·심리를 전망 근거로 전략에 반영하려는 경우.

## 분석 축

| 축 | 질문 |
|---|---|
| 세계정세 | 지정학, 제재, 무역, 공급망, 선거가 위험선호와 섹터에 어떤 영향을 주는가 |
| 경제상황 | 금리, 물가, 고용, 성장, PMI가 기대보다 강한가 약한가 |
| 정책/유동성 | 중앙은행, 재정, 국채발행, 달러 유동성이 위험자산에 우호적인가 |
| 환율/원자재 | 달러, 원/달러, 유가, 금속, 금이 섹터 이익과 외국인 수급에 어떤 영향을 주는가 |
| 투자자 심리 | 개인 과열/공포, 외국인·기관 수급, 선물 포지션이 지속될 가능성이 있는가 |
| 이벤트 캘린더 | 차주 발표와 실적, 옵션만기, 휴장이 가격 반응을 바꿀 수 있는가 |
| 숨은 기대치 | 공식 컨센서스와 달리 주가에 이미 반영된 기대치가 얼마나 높은가 |
| 선행 가격 반응 | NXT, ADR, 해외 동종주, 섹터 ETF, 선물시장이 먼저 경고하거나 선반영했는가 |

## 합성 절차

1. 근거 카드를 `risk_on`, `risk_off`, `mixed`, `neutral`로 분류한다.
2. 각 카드의 신뢰도, 최신성, 시장 전달 경로를 점수화한다.
3. 세계정세와 거시는 지수 방향, 환율, 금리, 원자재로 먼저 번역한다.
4. 수급과 심리는 외국인, 기관, 개인, 프로그램의 기대 포지션으로 나눈다.
5. 실적·상장·수주·정책 이벤트는 `공식 컨센서스`, `스트리트 기대치`, `주가 선반영 기대`, `발표 후 가격 반응 임계값`으로 나눈다.
6. 선행 시장과 정규장 검증을 분리한다. NXT, ADR, 프리마켓, 해외 동종주 반응은 관찰 재료이고, 정규장 VWAP·수급 확인 후 전략 근거가 된다.
7. 차주 전략은 기본 시나리오, 강세 시나리오, 약세 시나리오를 모두 만든다.
8. 각 시나리오에 확인 트리거와 무효화 조건을 붙인다.
9. 시나리오 안의 종목·ETF·ETN·ADR 후보는 `한글명/공식 풀네임 (티커)` 표시명으로 쓴다.
10. 전략 방향은 `공격`, `선별 공격`, `중립`, `방어`, `중단` 중 하나로 제한한다.
11. 내일/한 주 예측으로 이어지는 경우 각 시나리오마다 예측 대상, 필요한 최신 수치, 충돌 근거, 검증 시각을 남겨 `stock_predictive_calibration_skill.md`가 확률 보정을 수행할 수 있게 한다.

## 점수표

| 항목 | 점수 기준 |
|---|---|
| 출처 커버리지 | 필수 영역과 공식/1차 출처가 충분한가 |
| 최신성 | 기준일이 전망 기간에 맞는가 |
| 교차확인 | 핵심 주장에 2개 이상 독립 근거가 있는가 |
| 전달 경로 | 정보가 금리, 환율, 수급, 이익, 밸류에이션으로 연결되는가 |
| 심리 예측 | 개인·외국인·기관 심리가 분리되어 있는가 |
| 숨은 기대치 | 컨센서스와 주가 선반영 기대를 분리했는가 |
| 선행 시장 검증 | NXT, ADR, ETF, 해외 동종주와 정규장 가격 반응을 분리했는가 |
| 예측 보정 준비 | 확률화할 대상, 필수 데이터 수치, 검증 시각, 충돌 근거를 넘겼는가 |
| 시나리오 균형 | 기본/강세/약세와 무효화 조건이 있는가 |
| 전략 연결 | 후보군, 비중, 손절, 관찰 조건으로 이어지는가 |

## 판정

- `outlook_usable`: 85점 이상, 중대 차단 조건 0개. 전략 방향에 사용 가능.
- `conditional_outlook`: 75-84점. 보수적 비중과 조건부 관찰에만 사용.
- `watch_only`: 65-74점. 전망 메모로만 사용하고 신규 진입 근거로 쓰지 않음.
- `blocked`: 65점 미만 또는 중대 차단 조건 존재.

## 중대 차단 조건

- 검색·발췌 근거 없이 미래 전망을 작성한다.
- 차주 전략인데 당일/직전 거래일 가격, 환율, 금리, 수급을 확인하지 않는다.
- 세계정세 또는 뉴스 제목만으로 섹터 수혜/피해를 단정한다.
- 공식 컨센서스만 보고 이벤트 호재를 확정하고, 주가 선반영 기대와 발표 후 가격 반응을 분리하지 않는다.
- NXT, 프리마켓, ADR, 해외 동종주 반응을 정규장 수급 검증 없이 확정 매매 근거로 쓴다.
- 개인·외국인·기관 심리를 분리하지 않고 `수급 좋음/나쁨`으로만 쓴다.
- 공식 출처와 보조 출처 충돌을 무시한다.
- 기본/강세/약세 시나리오와 무효화 조건 없이 단일 방향을 단정한다.
- 내일/주간 예측 요청인데 확률 보정에 필요한 대상, 핵심 데이터 수치, 충돌 근거, 검증 시각을 남기지 않는다.
- 출처 기준일, 발표 일정, 다음 확인 시간이 없다.

## 출력

- 전망 점수와 판정.
- 기본/강세/약세 시나리오.
- 예측 보정 입력: 예측 대상, 핵심 데이터 수치, 충돌 근거, 검증 시각.
- 세계정세, 경제상황, 환율/금리/원자재, 투자자 심리별 핵심 근거.
- 지수, 섹터, 종목 후보군에 미치는 영향.
- 차주 전략 방향: 공격 / 선별 공격 / 중립 / 방어 / 중단.
- 확인 트리거, 무효화 조건, 다음 검색 시각.
