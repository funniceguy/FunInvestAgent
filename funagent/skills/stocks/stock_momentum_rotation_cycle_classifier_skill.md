# 스킬: 급등주·순환주 사이클 분류

## 목적

단기 강세 종목을 모두 같은 모멘텀으로 취급하지 않고, 급등 이벤트주·순환주·구조적 주도주·반등주·방어주·분산 국면을 확률로 분류한다. 각 유형의 서로 다른 지속 기간, 진입 시점, 매물 위험, 무효화 조건에 맞춰 전략을 선택한다.

## 적용 조건

- 단타, 스윙, 섹터 순환, 급등주, 주도주, 후발주, 눌림, 돌파 전략.
- 후보의 상승이 뉴스·숏커버·유동성 공백인지, 산업 사이클·섹터 자금 이동인지 구분해야 하는 경우.
- 급등 후 추격 여부나 순환매의 남은 구간을 판단하는 경우.

## 분류 체계

| 클래스 | 핵심 특징 | 기본 시간축 |
|---|---|---|
| parabolic_event_momentum | 단일 촉매, 비정상 수익률·RVOL, 좁은 유통물량, 높은 갭·일중 변동 | 분~3거래일 |
| rotational_cyclical | 섹터 폭 확장, 대장→후발 순환, 이익·가격 사이클, 기관성 수급 지속 | 수일~수주 |
| structural_trend_leader | 실적·추정치 상향과 장기 상대강도, 구조적 수요, 넓은 기관 보유 | 수주~수개월 |
| mean_reversion_rebound | 급락·과매도 뒤 숏커버·기술적 복원, 펀더멘털 확인 부족 | 장중~수일 |
| defensive_quality_rotation | 시장 위험회피 속 이익 안정·배당·저변동으로 이동 | 수일~수주 |
| distribution_markdown | 호재에도 약세, 고점 매물·매도 흡수, VWAP 이탈과 수급 악화 | 수일~수주 |
| hybrid_or_unknown | 두 클래스 이상 충돌 또는 핵심 데이터 부족 | 판정 보류 |

## 필수 증거 축

1. 가격: 1·3·5·20일 수익률, 갭, 장중 범위, 고점·저점 구조, VWAP/anchored VWAP.
2. 거래: RVOL, 거래대금, 유통주식 대비 회전율, volume-at-price, 장중 거래 집중도.
3. 폭과 상대강도: 섹터 ETF, 대장·중간·후발 종목 참여율, 지수 대비 상대강도, 신고가 확산.
4. 수급: 한국 외국인·기관·프로그램, 미국 ETF·경매·공시 포지션과 실제 체결, `supply_overhang_order_flow_packet`.
5. 촉매: 실적·가이던스·수주·정책·임상·M&A·리밸런싱과 공식 기대치·숨은 기대치.
6. 펀더멘털: 이익 추정 변화, 매출·마진·현금흐름, 산업 재고·가격·CAPEX 사이클.
7. 공급 위험: 유통물량, short interest의 기준일, 증자·ATM·전환증권·워런트·보호예수·내부자 매도.

## 판정 절차

1. 각 클래스에 0~100의 원점수를 부여하고 softmax 또는 정규화로 확률 합계 100%를 만든다.
2. 최고 클래스와 2위 클래스의 차이, 데이터 완결성, 상충 증거로 분류 신뢰도를 계산한다.
3. 가격·거래량만 있고 수급·촉매·펀더멘털이 없으면 신뢰도 상한을 60으로 둔다.
4. 독립 증거 축이 3개 미만이면 `hybrid_or_unknown`이다.
5. 미국과 한국의 세션·수급 자료 의미를 분리하고, 해외 선행 신호는 정규장에서 재검증한다.
6. 장전, 오픈 확인 구간, 10:30 전후, 종가에 재분류한다. 클래스가 바뀌면 기존 진입 논리를 자동 무효화하지 말고 변경 이유와 새 무효화 조건을 기록한다.

## 사이클 시계

### 급등 이벤트주

`ignition -> expansion -> exhaustion -> failed_bounce_or_base`

- ignition: 공식 촉매와 거래대금 급증, 첫 가격 발견.
- expansion: 고점 상승, VWAP 위, 매도 물량 흡수, 거래대금 지속.
- exhaustion: 거래대금은 크지만 고점 정체, 윗꼬리, 호가·체결 매도 흡수.
- failed_bounce_or_base: VWAP·전일 종가 회복 실패 또는 거래량 수축 기반 형성.

### 순환주

`accumulation -> leadership_broadening -> mature_rotation -> distribution`

- accumulation: 대장 상대강도와 기관성 수급 개선, 섹터 폭은 아직 좁음.
- leadership_broadening: ETF·대장·중간 종목으로 참여 확장, 이익 추정 또는 산업 지표 동행.
- mature_rotation: 후발주 급등, 밸류에이션 재평가, 거래 과열과 대장 둔화.
- distribution: 호재 반응 약화, 섹터 폭 축소, 대장 VWAP 이탈, 매도 오버행 증가.

## 전략 매핑

| 클래스 | 허용 전략 | 금지/제한 |
|---|---|---|
| parabolic_event_momentum | 첫 눌림·재돌파의 체결 확인, 매우 작은 단타 슬리브 | 장초반·애프터마켓 추격, 공급 공시 미확인 오버나이트 |
| rotational_cyclical | 섹터 폭·ETF·대장 확인 후 리더/준리더 분할 | 후발주만 급등한 말기 추격 |
| structural_trend_leader | 실적 논리 유지 시 눌림·기반 돌파 | 단기 약세만으로 구조 논리와 단타 논리 혼합 |
| mean_reversion_rebound | 전일 저점·VWAP 회복 후 제한 단타 | 낙폭만 보고 물타기, 추세 전환 단정 |
| defensive_quality_rotation | 시장 방어 슬리브와 상대강도 유지 | 시장 risk-on 전환 뒤 뒤늦은 추격 |
| distribution_markdown | 현금·축소·재진입 기준 재설정 | 호재 문구를 근거로 추가매수 |
| hybrid_or_unknown | 관찰과 추가 데이터 수집 | 공격적 방향 베팅 |

## 확률·비중 제한

- 분류 신뢰도 70 미만이면 클래스 기반 주방향 확률을 60% 초과로 올리지 않는다.
- parabolic_event_momentum의 exhaustion 또는 distribution_markdown이면 신규 매수는 `blocked`다.
- rotational_cyclical은 섹터 폭, 대장 상대강도, 수급 지속성 중 2개 이상 약화되면 `mature_rotation` 이상으로 보고 비중 확대를 금지한다.
- 급등주와 순환주 확률 차이가 15%p 미만이면 `hybrid`로 처리한다.
- 분류가 맞아도 `stock_supply_overhang_order_flow_skill.md`가 watch_only 이하이면 신규 진입을 허용하지 않는다.

## 출력

- `momentum_rotation_cycle_packet`
- 클래스별 확률 합계 100%, 최고 클래스, 2위 클래스, 신뢰도.
- 사용한 증거 축과 누락 데이터.
- 현재 사이클 단계와 다음 단계 전환 트리거.
- 급등주·순환주·구조적 주도주 판별 근거.
- 클래스에 맞는 보유 기간, 진입 창, 축소·무효화 조건.
- 최종 판정: `cycle_actionable`, `cycle_conditional`, `cycle_watch_only`, `cycle_blocked`, `unknown`.

## 중대 차단 조건

- 하루 급등이나 뉴스 제목만으로 급등주·순환주를 확정한다.
- 섹터 폭과 대장·후발 상대강도 없이 순환매를 단정한다.
- 유통물량·희석·매도 오버행 확인 없이 급등주 추격을 허용한다.
- 확률과 신뢰도 없이 단일 클래스를 정확한 판정으로 표현한다.
- 급등 이벤트주의 exhaustion을 구조적 주도주 눌림으로 바꿔 물타기를 허용한다.
- 순환주 말기 후발 급등을 사이클 초입으로 오판하고 비중을 확대한다.
- `hybrid_or_unknown` 또는 분류 신뢰도 60 미만인데 공격 전략을 제시한다.
