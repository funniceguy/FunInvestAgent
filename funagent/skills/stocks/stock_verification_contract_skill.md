# 스킬: 주식 Verify 평점 계약

## 목적

주식 전략, 자료 분석, 최종 리포트가 평점 기반 검증을 통과할 때까지 반복 평가하고 수정한다.

## 참조 규칙

- 공통 계약: `funagent/rules/verification_score_contract.md`
- 공통 체크리스트: `funagent/rules/verification_checklist.md`
- 주식 오케스트레이터: `stock_orchestrator.md`
- 심리 게이트: `stock_psychology_decision_gate_skill.md`

## 필수 적용 지점

- 데이터 수집 완료 후.
- 시장 국면 판단 완료 후.
- 단타/가치투자 전략 초안 완료 후.
- 포트폴리오 자금 배분 완료 후.
- 최종 리포트 작성 전.
- 성과 검토와 전략 재수정 전.

## 평가 절차

1. 평가 대상을 `data`, `analysis`, `strategy`, `portfolio`, `report`, `review` 중 하나로 분류한다.
2. 카테고리별 0-10점 항목 평가를 수행한다.
3. 100점 환산 점수와 전체 가중 점수를 계산한다.
4. 중대 차단 조건을 확인한다.
5. 기준 미달 항목을 수정한다.
6. 최소 2회, 최대 3회까지 반복 검증한다.
7. 통과하지 못하면 결론을 낮추고 부족한 데이터와 다음 액션을 리포트에 남긴다.

## 카테고리별 기본 항목

### 입력 적합성

- 목표와 기간이 명확한가.
- 투자 가능 자금과 제외 자금이 분리되었는가.
- 단타 손실 한도와 가치투자 기간이 분리되었는가.
- 비상금, 부채, 1년 내 확정 지출을 확인했는가.

### 데이터 신뢰도

- 가격과 거래 데이터가 당일 기준인가.
- 공시와 재무 데이터 기준일이 기록되었는가.
- 출처 ID가 공식 또는 1차 출처인가.
- 데이터 누락과 오래된 데이터가 표시되었는가.

### 상품·호가 실행 상세

- 실제 상품명, 티커, 거래소, 통화, 상품 유형이 기록되었는가.
- 현재가 또는 최근 종가, 최우선 매수호가, 최우선 매도호가, 호가 잔량, 기준 시각이 기록되었는가.
- 스프레드 금액, 스프레드 %, 거래량, 거래대금, 20일 평균 거래대금 또는 유동성 대체 지표가 기록되었는가.
- ETF/ETN은 NAV/IOPV, 괴리율, 총보수, 시장조성 상태가 확인되었는가.
- 호가 미확인 또는 지연 데이터일 때 신규 주문 결론을 낮췄는가.

### 투자자 심리 해석

- 정보, 기대치, 포지션, 가격 반응, 수급 지속성 5단계를 모두 해석했는가.
- FOMO, 공포, 본전 심리, 처분효과, 과잉확신 중 해당 플래그를 표시했는가.
- 개인 과열과 기관/외국인 수급 지속성을 분리했는가.
- 심리 게이트가 `watch_only` 또는 `avoid`일 때 신규 진입을 차단했는가.

### 분석 논리

- 시장 국면 판단 근거가 충분한가.
- 후보 선정 기준과 제외 기준이 분리되었는가.
- 반대 시나리오가 있는가.
- 단타와 가치투자 근거가 섞이지 않았는가.

### 전략 완성도

- 단타 진입, 손절, 청산 조건이 있는가.
- 가치투자 매수, 보유, 논리 훼손 조건이 있는가.
- 슬리브별 자금 배분이 명확한가.
- 실행하지 말아야 할 조건이 있는가.

### 리스크 통제

- 종목, 섹터, 테마 집중 한도가 있는가.
- 세금, 수수료, 환율, 유동성을 반영했는가.
- 법률/규제 이슈를 확인했는가.
- 미수, 신용, 레버리지 사용 여부가 통제되었는가.

### 레버리지 상품 적합성

- 레버리지 상품 구조와 일일 리셋을 확인했는가.
- 괴리율, 유동성, 스프레드, 비용을 확인했는가.
- 손절가, 하루 손실 한도, 최대 보유 시간이 명확한가.
- 3배 레버리지 게이트가 `allowed` 또는 `conditional_allowed`인가.
- 손실 복구, 물타기, 오버나이트 갭 리스크가 통제되었는가.

### 성과 검토성

- 벤치마크가 있는가.
- 성공/실패 기준이 수치화되었는가.
- 단타와 가치투자의 검토 주기가 다르게 정의되었는가.
- 실패 원인 분류와 다음 수정 규칙이 있는가.

## 출력

- 회차별 Verify 표.
- 기준 미달 항목.
- 수정 내역.
- 최종 평점.
- 최종 판정.
