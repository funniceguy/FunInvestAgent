# 스킬: 주식 투자 범위 정의

## 목적

주식 전략을 만들기 전에 사용자 상황과 전략 범위를 확정한다.

## 입력

- 기준 통화와 관할권.
- 투자 가능 총자금.
- 단타에 사용할 수 있는 최대 손실 가능 자금.
- 공격적 단기 매매 허용 여부, 1회 손실 한도, 장중 실시간 대응 가능 시간.
- 가치투자 목표 기간.
- 전망 기반 전략이면 전망 기간: 다음 주, 1개월, 분기, 6-12개월.
- 현재 보유 종목과 비중.
- 비상금, 부채, 1년 내 확정 지출.
- 시장 범위: 한국, 미국, 글로벌, 특정 섹터.

## 절차

1. 사용자 요청을 `단타`, `공격적 단기`, `가치투자`, `전망 기반`, `혼합 운영`, `성과 검토` 중 하나 이상으로 분류한다.
2. 실제 금액이 없으면 비중 기반 보고서로 전환한다.
3. 단타 슬리브와 장기 슬리브를 분리한다.
4. 투자 제외 자금을 먼저 표시한다.
5. 공격적 단기 매매는 대응 가능 시간, 1회 손실 한도, 최대 하루 손실 한도가 없으면 `보류`로 제한한다.
6. 필수 정보가 없으면 전략 결론을 `조건부 검토` 또는 `보류`로 제한한다.

## 출력

- 전략 범위.
- 투자 가능 자금과 제외 자금.
- 손실 허용 한도.
- 필요한 추가 질문.
