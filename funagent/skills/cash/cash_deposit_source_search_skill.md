# 스킬: 현금성·예적금 공식 정보 검색

## 목적

예금성 상품의 금리·약관·우대조건·보호 여부를 공식 비교공시와 금융회사 원문에서 수집하고, 채권 비교에 필요한 같은 기준일 자료를 연결한다.

## 검색 순서

1. 금융당국·예금보험기관의 최신 보호 대상, 보호 금융회사, 한도와 시행일.
2. 금융감독기관·금융협회의 금융상품 비교공시.
3. 해당 금융회사의 상품설명서·약관·금리표·예금보험관계 표시.
4. 중앙은행의 신규취급액 수신금리와 시장금리.
5. 비교 채권의 국채 수익률곡선, YTW, 가격·호가·수수료.

## 필수 데이터

- 상품명, 금융회사 법인명, 상품 유형, 통화, 가입 대상·채널.
- 기본금리, 최고우대금리, 우대조건별 가산폭, 적용 잔액·월 납입 한도.
- 금리 기준일, 변동 가능 여부, 프로모션 종료일, 자동재예치와 만기 후 금리.
- 중도해지금리·부분출금·이체 제한·가입 기간.
- 보호 대상 여부, 보호 금융회사, 보호 한도 기준일, 동일 금융회사 기존 보호 예금 합계.
- 세율·비과세·우대세율은 계좌·거주자 조건과 최신 공식 근거.

## 출처 계약

- 비교공시는 후보 탐색에 사용하고 가입 전 금융회사 원문으로 재확인한다.
- 광고·앱 화면·검색 요약의 최고금리는 원문 약관과 불일치하면 전략 근거로 쓰지 않는다.
- `data_as_of`, `published_at`, `checked_at`, URL과 변경 가능성을 기록한다.
- 현재 조건을 확인하지 못하면 `unknown`으로 남기고 최고금리·보호·가입 가능을 추정하지 않는다.

## 출력

- 출처 커버리지와 누락·충돌 자료.
- 상품별 공식 문서와 적용 기준일.
- 보호 대상·한도·금리·우대조건의 evidence_id.
- 다음 금리 갱신 또는 프로모션 종료 확인일.
