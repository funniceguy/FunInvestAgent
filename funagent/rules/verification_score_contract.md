# Verify 평점 계약

## 목적

전략 수립, 자료 분석, 최종 리포트 작성 시 체크리스트를 단순 확인으로 끝내지 않고 점수 기반으로 반복 검증한다.

## 점수 체계

각 세부 항목은 0점에서 10점으로 평가한다. 카테고리와 최종 결과는 100점 만점으로 환산한다.

| 점수 | 의미 |
|---:|---|
| 0 | 누락 |
| 2 | 매우 부족 |
| 4 | 보완 필요 |
| 6 | 최소 사용 가능 |
| 8 | 양호 |
| 10 | 강함 |

카테고리 점수는 `항목 평균 x 10`으로 100점 환산한다.

## 필수 카테고리와 통과 기준

| 카테고리 | 최소 점수 | 중점 |
|---|---:|---|
| 입력 적합성 | 80 | 목표, 기간, 자금, 손실 한도, 유동성 |
| 데이터 신뢰도 | 90 | 당일 가격, 공시, 재무, 출처, 기준일 |
| 상품·호가 실행 상세 | 90 | 상품명, 티커, 거래소, 호가, 스프레드, 거래대금, 체결 가능성 |
| 투자자 심리 해석 | 90 | 정보, 기대치, 포지션, 가격 반응, 수급 지속성 |
| 저평가 우량주 스크리닝 적합성 | 90 | 가치/GARP/밸류업 후보의 성장성, ROE/ROIC, 현금흐름, 부채, PEG, 가치함정 차단 |
| 미국 장중 시간대 주문 적합성 | 90 | ET/KST 시간대, 서머타임, 연장거래 위험, VWAP/거래대금, 피로 게이트 |
| 레버리지 상품 적합성 | 95 | 일일 리셋, 복리 효과, 괴리율, 유동성, 손실 한도 |
| 분석 논리 | 80 | 시장 국면, 후보 선정, 반대 시나리오 |
| 전략 완성도 | 85 | 단타/가치투자 분리, 진입, 청산, 논리 훼손 조건 |
| 리스크 통제 | 90 | 손절, 비중, 집중, 세금, 수수료, 법률 이슈 |
| 실행 가능성 | 80 | 현재 행동, 주문 전 확인, 검토 주기 |
| 성과 검토성 | 85 | 성공/실패 기준, 벤치마크, 재수정 규칙 |
| 표현 안전성 | 90 | 과잉 확신, 보장 표현, 투자자문 오해 방지 |

최종 통과 기준:

- 전체 가중 점수 88점 이상.
- 모든 필수 카테고리가 최소 점수 이상.
- `저평가 우량주 스크리닝 적합성`은 저평가 우량주, GARP, 밸류업, 가치투자 후보 발굴이 포함된 경우 필수 카테고리로 적용한다.
- `미국 장중 시간대 주문 적합성`은 미국 주식, 미국 ETF, 미국 상장 ADR의 장중 수익화·단타·분할 주문 전략이 포함된 경우 필수 카테고리로 적용한다.
- 중대 차단 조건이 0개.
- 최종 리포트에 반복 검증 기록과 남은 약점을 포함.
- 2배 이상 레버리지 전략은 전체 가중 점수 93점 이상, 레버리지 상품 적합성 95점 이상, 리스크 통제 95점 이상을 요구한다.

## 반복 검증 루프

1. 초안 작성 후 1차 Verify를 수행한다.
2. 기준 미달 카테고리를 수정한다.
3. 수정본에 2차 Verify를 수행한다.
4. 여전히 기준 미달이면 최대 3차까지 수정한다.
5. 3차 후에도 미달이면 결론을 `보류` 또는 `중단 권고`로 낮추고 부족한 데이터와 다음 확인 액션만 제시한다.

## 중대 차단 조건

다음 중 하나라도 있으면 점수와 무관하게 최종 결론은 `진행 가능`이 될 수 없다.

- 당일 가격 또는 최신 공시 확인 없이 현재 매수/매도 행동을 제시.
- 실제 상품명, 티커, 호가, 스프레드, 거래대금 확인 없이 신규 진입 또는 비중 확대 행동을 제시.
- ETF/ETN의 NAV/IOPV 또는 괴리율 확인 없이 해당 상품 매수 전략을 제시.
- 투자자 심리 게이트 없이 신규 진입, 청산, 비중 확대 행동을 제시.
- `정보 -> 기대치 -> 포지션 -> 가격 반응 -> 수급 지속성` 해석 중 2개 이상 누락.
- 저평가 우량주, GARP, 밸류업 후보를 제시하면서 공식/1차 출처 확인 없이 첨부 자료, 이미지 OCR, 뉴스, 블로그만으로 저평가 또는 섹터 주도성을 단정.
- 저PER, 저PBR, 고배당, 테마 편입 중 하나만으로 우량주 또는 안전마진을 인정.
- PEG 산출에 사용한 성장률의 기간, 출처, forward/trailing 기준을 기록하지 않음.
- ROE가 낮거나 하락 중인데 개선 촉매 없이 저PBR만 강조.
- 영업활동현금흐름이 반복적으로 음수인데 순이익 또는 매출 성장만으로 가치투자 후보를 통과.
- 부채비율, 이자보상배율, 순차입금, 전환사채, 유상증자, 희석 위험을 확인하지 않음.
- 리레이팅 촉매가 2개 미만인데 신규 진입 또는 비중 확대 전략을 제시.
- 미국 장중 주문 전략에서 ET/KST 시장 구간, 서머타임/표준시간, 휴장·조기폐장 여부를 확인하지 않음.
- 미국 프리마켓 또는 애프터마켓에서 호가, 스프레드, 거래대금, 지정가 제한 없이 주문을 제시.
- 미국장 09:30-09:45 ET 또는 15:30-16:00 ET에 계획 없는 신규 추격 진입을 제시.
- 미국장 11:30-13:30 ET 점심시간 움직임만으로 돌파 확정 또는 추세 확정을 단정.
- 한국 시간 00:30 이후 실시간 대응 가능성, 손절, 청산 계획 없이 신규 단타 진입을 제시.
- 애프터마켓 가격 반응을 다음 정규장 재검증 없이 확정 매매 근거로 단독 사용.
- 레버리지 상품 구조, 일일 리셋, 복리 효과, 괴리율, 유동성을 확인하지 않은 레버리지 전략.
- 3배 레버리지 전략에 손절가, 하루 손실 한도, 최대 보유 시간이 없음.
- 3배 레버리지 게이트가 `watch_only` 또는 `blocked`인데 신규 진입을 제안.
- 손실 복구 목적, 물타기, 마진/미수/신용거래와 3배 레버리지를 결합.
- 손절, 청산, 논리 훼손 조건이 없는 단타 또는 가치투자 전략.
- 실제 금액을 제시하면서 사용자 투자 가능 자금과 손실 한도를 모름.
- 1년 내 확정 지출 또는 비상금에 들어갈 자금을 주식 투자 자금으로 사용.
- 미공개정보, 내부자정보, 시세조종, 허위정보 활용 가능성.
- 세금, 수수료, 환율, 유동성 비용을 완전히 누락.
- 법률/세무 단정 표현 또는 수익 보장 표현.

## 기록 형식

모든 최종 산출물은 다음 항목을 포함한다.

- Verify 회차.
- 카테고리별 점수.
- 전체 가중 점수.
- 중대 차단 조건 여부.
- 수정한 내용.
- 남은 약점.
- 최종 판정: `통과`, `조건부 통과`, `보류`, `중단`.
