# 검증 체크리스트

## 평점 계약

- [ ] 각 항목을 0-10점으로 평가했는가.
- [ ] 카테고리 점수를 100점 환산으로 기록했는가.
- [ ] 전체 가중 점수와 통과 기준을 비교했는가.
- [ ] 최소 2회 이상 반복 검증했는가.
- [ ] 기준 미달 항목의 수정 내역을 기록했는가.
- [ ] 중대 차단 조건 발생 여부를 기록했는가.
- [ ] 최종 판정이 점수와 차단 조건에 맞게 낮춰졌는가.
- [ ] 차주/미래 전망 전략의 검색·발췌·분석 적합성 점수가 기준 미달이면 진행 가능으로 표시하지 않았는가.
- [ ] 투자자 심리 게이트를 통과하지 못한 전략을 진행 가능으로 표시하지 않았는가.
- [ ] 공격적 단기 매매 적합성 점수가 기준 미달이면 신규 진입을 진행 가능으로 표시하지 않았는가.

## 입력 검증

- [ ] 사용자 목표가 명확한가.
- [ ] 투자 기간이 명확한가.
- [ ] 위험 감내도가 기록되어 있는가.
- [ ] 비상금과 1년 내 확정 지출을 분리했는가.
- [ ] 부채와 금리를 확인했는가.

## 데이터 검증

- [ ] 가격, 금리, 법령, 공시는 최신 출처로 확인했는가.
- [ ] 데이터 기준일을 적었는가.
- [ ] 출처 ID를 기록했는가.
- [ ] 추정과 사실을 분리했는가.
- [ ] 차주/미래 전망에 사용한 출처의 source_id, published_at, data_as_of, source_checked_at을 기록했는가.
- [ ] 실제 상품명, 티커, 거래소, 통화, 상품 유형을 기록했는가.
- [ ] 최우선 매수호가, 최우선 매도호가, 호가 잔량, 스프레드 %, 기준 시각을 기록했는가.
- [ ] 거래량, 거래대금, 20일 평균 거래대금 또는 유동성 대체 지표를 기록했는가.
- [ ] 저평가 우량주/GARP/성장 품질 후보의 OPM 또는 금융 업종별 대체 수익성 지표를 기록했는가.
- [ ] ETF/ETN이면 NAV/IOPV, 괴리율, 총보수, 시장조성 상태를 기록했는가.
- [ ] 호가 미확인 또는 지연 데이터일 때 신규 주문 결론을 낮췄는가.

## 전략 검증

- [ ] 보수, 기준, 공격 시나리오를 구분했는가.
- [ ] 차주/미래 전망은 기본/강세/약세 시나리오와 무효화 조건을 포함했는가.
- [ ] 세계정세와 경제지표를 금리, 환율, 원자재, 수급, 이익, 밸류에이션, 위험선호 전달 경로로 연결했는가.
- [ ] 실행 조건과 중단 조건이 있는가.
- [ ] 상품별 주문 방식, 지정가 기준, 체결 가능성 판정이 있는가.
- [ ] 세금, 수수료, 환율, 유동성 비용을 언급했는가.
- [ ] 단일 자산 집중 리스크를 확인했는가.
- [ ] 공격적 단기 매매의 기대치, 가격 반응, 거래대금, 기관/외국인/프로그램 수급, 환율, 상대강도, 손익비를 점수화했는가.
- [ ] 수급과 환율이 단기 방향성과 충돌하면 watch_only 또는 blocked로 낮췄는가.
- [ ] 저마진, 낮은 ROE, 약한 영업현금흐름, 부채·이자 부담, 성장 촉매 부재, 주주환원 부재, 약한 수급 7개 위험 신호를 카운트했는가.
- [ ] 위험 신호 4개 이상 후보를 랭킹에서 제외하고, 2-3개 후보는 보류/관찰로 낮췄는가.

## 투자자 심리 검증

- [ ] 정보, 기대치, 포지션, 가격 반응, 수급 지속성 5단계를 해석했는가.
- [ ] 정보 등급을 A-E로 분류했는가.
- [ ] FOMO, 공포, 본전 심리, 처분효과, 과잉확신, 제한된 주의 중 관련 플래그를 표시했는가.
- [ ] 호재/악재 대비 가격 반응의 역설을 검토했는가.
- [ ] 개인 과열과 기관/외국인 수급 지속성을 분리했는가.
- [ ] 환율 순풍/역풍이 외국인 수급과 단기 가격 반응에 미치는 영향을 분리했는가.
- [ ] 미래 수급 전망에서 개인, 외국인, 기관, 프로그램 심리를 따로 기록했는가.
- [ ] 장초반 추격, 시간외 급등 추격, 긴 윗꼬리 분산 위험을 검토했는가.
- [ ] 심리 게이트 평균 점수와 중대 위험을 기록했는가.

## 레버리지 검증

- [ ] 레버리지 상품의 일일 리셋과 장기 성과 괴리 가능성을 설명했는가.
- [ ] 괴리율, 유동성, 스프레드, 비용, 세금을 확인했는가.
- [ ] ETN이면 발행사 신용위험과 조기상환/상장폐지 조건을 확인했는가.
- [ ] 3배 레버리지 게이트 점수와 판정을 기록했는가.
- [ ] 손절가, 하루 손실 한도, 최대 보유 시간을 사전에 정했는가.
- [ ] 오버나이트 갭 리스크를 원칙적으로 차단하거나 별도 계획을 썼는가.
- [ ] 손실 복구, 물타기, 마진/미수/신용거래 결합을 금지했는가.

## 법률 및 컴플라이언스 검증

- [ ] 관할권을 확인했는가.
- [ ] 상품 유형과 행위 유형을 분류했는가.
- [ ] 법률/세무 단정 표현을 제거했는가.
- [ ] 전문가 검토 필요 여부를 표시했는가.

## 출력 검증

- [ ] 요약이 명확한가.
- [ ] 핵심 리스크가 앞부분에 있는가.
- [ ] 출처와 기준일이 포함되어 있는가.
- [ ] 다음 액션이 구체적인가.
