# 리스크 점수 모델

## 목적

투자 판단을 감정이나 분위기에 맡기지 않고, 비교 가능한 기준으로 위험을 표시한다.

## 점수 범위

각 항목은 0점에서 10점으로 평가한다.

| 점수 | 의미 |
|---:|---|
| 0 | 해당 없음 |
| 2 | 낮음 |
| 4 | 관리 가능 |
| 6 | 주의 |
| 8 | 높음 |
| 10 | 매우 높음 |

## 항목

- 시장 변동성.
- 금리 민감도.
- 신용 위험.
- 환율 위험.
- 유동성 위험.
- 레버리지.
- 집중도.
- 상품 복잡도.
- 세금 불확실성.
- 법률 불확실성.

## 종합 등급

| 총점 | 등급 | 해석 |
|---:|---|---|
| 0-20 | 낮음 | 일반적인 모니터링으로 충분 |
| 21-40 | 보통 | 비중 제한과 재검토 주기 필요 |
| 41-60 | 높음 | 실행 전 추가 확인 필요 |
| 61-80 | 매우 높음 | 보수적 접근 또는 대안 검토 |
| 81-100 | 중단 권고 | 전문가 검토 전 실행 보류 |

## 강제 상향 조건

다음 중 하나라도 해당하면 종합 등급을 최소 `높음`으로 올린다.

- 사용자가 1년 내 필요한 자금을 투입하려 한다.
- 레버리지 또는 파생 구조가 포함된다.
- 원금 손실 가능성을 이해하지 못했다.
- 법률 또는 세금 효과가 수익성의 핵심이다.
- 총자산의 30% 이상이 단일 자산에 집중된다.
